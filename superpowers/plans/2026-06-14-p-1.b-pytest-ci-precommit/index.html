    For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

    Goal: Establish pytest test framework, GitHub Actions CI (3 workflows), and pre-commit hooks on top of P-1.A's project metadata baseline.

    Architecture: Sequential PRs with state-sensitive steps. P-1.A established pyproject.toml with dev deps; P-1.B activates them via test framework + CI + pre-commit. test_integrity.py is migrated unittest→pytest (preserving coverage). All hooks/configs are additive.

    Step 1.6: Create tests/unit/test_coverage_thresholds.py

    • [ ] Create with exact content:
    """Enforce coverage thresholds post-test.

Branch coverage threshold enforcement via pytest (since --cov-fail-under
only applies to line coverage). Runs LAST (via @pytest.mark.last) to ensure
coverage.xml has been generated by earlier tests.
"""
import xml.etree.ElementTree as ET
from pathlib import Path

import pytest

COVERAGE_XML = Path("tests/coverage/coverage.xml")
LINE_THRESHOLD = 90
BRANCH_THRESHOLD = 80


@pytest.mark.last
def test_branch_coverage_meets_threshold():
    """Branch coverage must be >= 80%.

    Runs last to ensure coverage.xml exists. CI must split pytest into
    `-n auto -m "not last"` (parallel main) and `-p no:xdist -m "last"`
    (serial threshold check).
    """
    if not COVERAGE_XML.exists():
        pytest.fail(
            "coverage.xml not found; ensure pytest runs with "
            "'--cov-report=xml:tests/coverage/coverage.xml' and "
            "test_coverage_thresholds runs AFTER all other tests"
        )

    tree = ET.parse(COVERAGE_XML)
    root = tree.getroot()

    total_branches_valid = 0
    total_branches_covered = 0
    for cls in root.findall(".//class"):
        for counter in cls.findall("counter"):
            if counter.get("type") == "BRANCH":
                total_branches_valid += int(counter.get("valid"))
                total_branches_covered += int(counter.get("covered"))

    if total_branches_valid == 0:
        pytest.skip("No branches in coverage data")

    branch_pct = (total_branches_covered / total_branches_valid) * 100
    assert branch_pct >= BRANCH_THRESHOLD, (
        f"Branch coverage {branch_pct:.1f}% below threshold {BRANCH_THRESHOLD}%. "
        f"Covered {total_branches_covered}/{total_branches_valid} branches."
    )

    Step 2.2: Migrate unittest → pytest

    The migration pattern: - class TestXxx(unittest.TestCase) → drop class, use module-level functions - self.assertEqual(a, b) → assert a == b - self.assertTrue(x) → assert x - self.assertIn(a, b) → assert a in b - self.assertRaises(E) → with pytest.raises(E): - self.setUp() / self.tearDown() → pytest fixtures (@pytest.fixture) - setUpClass / tearDownClass → module-level or class-scoped fixtures - Drop if __name__ == "__main__": unittest.main() - Drop import unittest if no longer needed
