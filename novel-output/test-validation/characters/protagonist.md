# 陆衡 · 主角深度档案

> 主角弧线单一权威：成长弧线只写于此文件，story_bible 与其他文件均以引用为准，不得复述。

## 一、来历与身份

陆衡，二十四岁，原世界某高校材料科学与工程博士在读，研究方向是极端条件下晶体缺陷与能量输运。一场实验室深夜的异常放电之后，他在灰港的咸风与鱼腥味里醒来，全身家当只剩一件起球的实验服、一支记号笔、一本被汗浸透的实验记录本，以及脑子里一套完整的现代科学训练。

他不懂烬洲的任何咒文，不会借取源息，最初三天连当地方言也听不全，被灰港码头的人当成"被海风吹坏了脑子的北地蛮子"。但他有一样所有源贵都没有的东西：一双被训练来"先测量、再下结论"的眼睛。

## 二、性格内核

陆衡不是热血型主角。他的第一反应永远是观察、记录、建立假设，而不是拔刀或立誓。这种气质让他在崇尚"立血契、赌气运"的烬洲显得格格不入，也让他屡屡被误判为懦弱——直到人们发现，他那一本皱巴巴的记录本上，已经把对方三次施法的源息流向、借还比、反噬征兆画成了图。

| 维度 | 表现 |
|------|------|
| 认知方式 | 一切命题必须可被独立复现，否则只是"尚未被证伪的猜测" |
| 处事倾向 | 优先谈判与交换，把冲突转化为可量化的博弈 |
| 道德底线 | 不以"天意"为恶行辩护；不立自己无法兑现的血契 |
| 致命弱点 | 过度相信"只要证据足够，所有人都会接受真相"——这在圣座治下是天真 |

他的核心张力在于：他笃信知识应当普惠，却身处一个以知识垄断为根基的神权秩序。他每公开一个发现，都是在拆圣座的合法性。

## 三、能力与限制（与世界法则接口）

- **源息浓度近乎素民**：短期内无法直接施法，前中期不是战斗型主角。
- **观测即武器**：他把实验记录本改造成"源息账本"，逐次记录施法的借取量、还息比、反噬征兆，逐步建立这个世界第一张"能量流图"。
- **方法论降维**：可证伪性、对照实验、能量守恒——这套工具在源息世界近乎降维，但也使他成为圣座最危险的异端。
- **与世界铁律的全面相撞**：不立血契（律五）、不入学派（律三）、不议灾变成因（律十）——他几乎每一条生存法则都在与世界硬律相悖。

## 四、与世界的核心冲突

- 对**圣座**：他是散布"谬说"的异端，须以烬刑 silenced（律四、律十）。
- 对**赤霞王朝**：他揭示的源脉衰竭真相，将引爆三位王子的承继之争（律六）。
- 对**铁冠诸邦**：他的源息账本若公开，将动摇灰铁之主南下掠脉的战争理由。
- 对**地下密会**：他既是被追捧的"还原先民之学"之钥匙，也是一颗可能把密会推向真正异端（镜像三禁，律七）的危险引信。
