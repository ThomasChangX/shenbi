# 界

透气孔中的光柱截面落在地面上的位置比昨日早了约一掌半。

他在黑暗中睁开眼。眼睑在光线到达之前完成了开合——身体在醒来后的第一个动作是确认光柱的位置。不是看时间。是身体需要一个参照点来判断一天在睡眠中向前移动了多少。光柱偏移了一掌半。比昨日早。说明他在更早的时间醒来——身体的睡眠时长在缩短。代谢需求在增加，恢复时间在减少。

坐起来。铁梯第三阶在他踩上去时发出一个与昨日不同的吱呀角度。不是幅度变形量不同——是身体重心在蹬踏时偏向前脚掌，推力的方向比昨日更朝前。他在地下室站起来的角度已经和前几日不同，重心在坐姿转换为站姿的过程中全程靠前。

手机。拿起时背壳的温度——比昨天早晨高了一个可辨别的温差。大约一度半到两度。夜间体温没有回落到前日基线。身体在睡眠状态下的热量输出上移了。触发段的跃升没有在八小时的恢复中消散。

蹲下。门框前。

闭眼。感知从手指向木框表面延伸。左边缘偏下——昨天早晨的初始均匀态起点已经偏移了。热辐射梯度在一夜的重新分布后出现了一个积累方向。它偏离了昨天的起点，偏向右侧约一指宽度。温度分布轮廓线在这个位置形成了一个倾斜的坡度。

积累的起始方向与上一周期相同。方向可重复性确认。偏移量在第一天(白纸起点)→第二天(可见单侧热累积)的增量在预期范围内——约为上一周期同段位的65%到70%，不是百分百重复。有差异。积累不是机械复制——每次的底色环境参数(外部气温/湿度/林烽自身的体温基线变化/门板含水率)不同。

他睁开眼。站起来。推门。

今天门廊前的空间在他的脚掌接触到第一块地面砖时传递了一个不同昨天的振动反馈。地面砖的湿度不同——夜间露水比昨日重。空气在门打开后流过门廊时的流速——比昨日快了约一档。风的方向在早晨阶段已经发生偏移。

他走出去。

-----

窄巷分叉处。他站在第一条窄巷入口前时，在距离入口约三米的位置停住了。

入口的路面颜色不对。不是地面砖的色差——是路面上的鞋印密度和方向在那个位置形成了一个异常分布。晨间露水还在地面砖上，鞋印在露水层上留下了清晰的轮廓——不只有他的。有人在今天更早的时间经过了这个入口。方向是进去的，返回的鞋印密度比进入的少。

他在三米外停住。

停住的动作不是由意识启动的。身体在眼睛完成"鞋印密度异常"的判断前已经减速。左脚在前、右脚跟上的步伐序列在第二步落地的时长上延长了——步幅缩短、触地时间延长。身体在视觉确认前已经进入了警告状态下的减速模式。

他停在入口外。没有进入。

窄巷入口在前方。连续数日走了同一条路线——第五天的身体自动转向、第六天(今日)在入口前停住。停住不是因为看见了具体的危险——是因为路面上的信息在一个维度上偏离了日常模式。昨天之前从没见过其他人的鞋印在晨间时段出现在这条巷子入口。

他在入口外保持了约四次呼吸的停顿。入口末端的声反射——窄巷另一头的声音在这个时间段应该只有风在墙缝中通过时的低频流动声。今日——除了风声，还有一个低频人声的残余。不是说话声，是咳嗽或清嗓子的声音在窄巷中段被墙体反射后传到入口的残余波形。有人在窄巷中段的位置。今天更早到达了那里。

他退后半步。

退后的位置使他从入口的直射视线中偏移了一个偏角。视线穿过入口的三角形视野范围——窄巷内约十二到十五米深处的路面在晨光中可见。没有人在那里。路面在可见范围内没有异常。但不可见的分叉口在十五米外——声音从那个方向来。

他在入口外停留的时间比前五天加在一起在入口处停留的时间都长。

两秒后他转身。

转身时他知道了自己为什么转身。不是因为窄巷不安全——是因为他前五天积累的"窄巷=安全"等式中出现了一个从未出现的变量: 入口在早晨就有人来过。他是第六天来走这条路的人。连续五天的独享路径变成了一条另有使用者的路。安全的条件——不被注意——在这个入口处已经不成立了。

他的身体在做出转身决定时没有任何犹豫。转身后他走了另一个方向——沿宽路边缘向南偏西走了一段，在一个没有名称的岔口向东折入一条更窄、更脏、地面铺着焦炭渣的过道。

这条过道他走过一次。第五天时作为窄巷网络的备用选项在脑子里标记过。

-----

过道。宽度比那条窄巷窄了约三分之一。两侧墙体不是砖石——是废料堆叠形成的边界，铁皮和木板在多年的暴露中形成了稳定的堆积体。高度平均的间隙在顶部透入狭长的天光。地面焦炭渣的粒度分布不均匀——脚踩上去的反馈不稳定，有些位置压实、有些位置松散到脚掌会下沉约半指深度。

他在过道中段停下来。墙边有一个被铁桶和木板遮蔽的三角形空间。

取出MH-020。

屏幕亮起。折线图标的位置——在屏幕的左下区域，比昨日的左上位置偏了下。波形在折线图标区域的展开——不是单条连续波形，也不是多线叠加。是三条短波形，在屏幕左下区域从同一原点辐射。折线角度不同——一条偏左、一条偏右、一条下行——形成一个扇形展开的波形组。

屏幕下方浮出数字。十二点一。十一点八。十二点四。

格式串。数字下方紧随着浮出一行:
[s] 12.1 / 11.8 / 12.4

他在铁桶遮蔽的阴影中保持蹲姿。持握角度——屏幕在这条过道的方向上——折线图标在扇形波形组的指向与过道的走向呈一个约五十度的夹角。昨天在窄巷中同一MH-020读出的数值在13.3到12.7之间。今天是11.8到12.4。基线偏移了约一个单位。

他把手机保持在操作模式，蹲着走了两步——从铁桶遮蔽的阴影移到墙边另一侧的光线可以从上方进入的位置。

折线图标在屏幕上的位置偏移了。原来是左下，现在向左上回移了一段——约等于屏幕宽度的十分之一。波形组的展开角度在这个位置变窄了——三条短折线的间距压缩了约两个像素。扇形束聚拢了。

屏幕下方的数值: 十二点八。十二点一。十二点五。格式串: [n] 12.8 / 12.1 / 12.5。

尾缀变了。刚从[s]换到了[n]。

他退回阴影位置。数值回到刚才的范围。尾缀回到[s]。

他保持蹲姿约三四次呼吸的时间。屏幕在同一个位置读了两轮——数值和尾缀在阴影位置稳定，在光照位置跳变。

换了个地方读出来的数不一样。不是设备故障——是环境参数对测量内容的修正。他从阴影到光照位置移动了约一米，屏幕上的折线图标位置偏移了、波形展开角度变化了、数字范围整体上移了、尾缀变了。

他把手机收起来。蹲着的时间在这条过道中——从取出到收起大约经历了八到十次呼吸。不算长，但在一条宽度不足一臂的新过道中暴露了操作的全部时间。如果有人走过这条过道的入口，他在铁桶后面不一定在第一时间被看见，但如果有人在入口停住——他的位置在遮蔽物的深度上不够远。

他站起来。

-----

过道尽头汇入一条更宽的路面——地面痕迹显示这里在近期被车轮碾压过多次。车辙深度不均，碾过的方向是偏东偏北。

他在出口前停下来。

路面的开阔度比之前的过道大——两侧建筑间距约六到七米，足够一辆中型车辆通过。天空在这一段的开阔面上露出了一片偏灰的白——云层比前几日厚。光线在云层中的散射使路面上的阴影边界模糊，没有清晰的轮廓线。

他在开阔边缘的位置再次取出MH-020。

屏幕显示与在过道中两次都不同。折线图标在屏幕中央偏右的位置——不是左下也不是左上。波形在折线图标区域不是扇形束——是单条逐渐衰减的波形，振幅从左侧峰值向右衰减，在大约屏幕宽度三分之二处归零。

数值: 十三点零。九点七。十一。格式串: [d] 13.0 / 9.7 / 11.0。

尾缀是[d]。

窄巷封闭空间→[s]。半开敞堆叠边界→[n]。开阔边缘→[d]。

他在开阔边缘站了约两次呼吸。看屏幕。扇形束变成单条衰减波形——环境参数改变了显示模式。尾缀从[s]到[n]到[d]——对应位置的物理开敞度。

他收起手机。收起的动作比在过道中快。开阔边缘在空旷路面的边缘——可遮蔽物较少，操作时间需要更短判断更果断。这是信息的成本。在阴影位置可以读八次呼吸的信息——但在开阔边缘只能读两次呼吸。想看更多就得冒更多风险。今天他已经验证了: 同一个设备在不同位置读出的数不一样、尾缀和位置的物理参数有关系、开阔地可用的信息获取时间比封闭空间短。

他在开阔边缘沿着一段低矮的砖墙向西偏南方向移动。墙高约齐腰，墙根处长着半枯的草，草叶上露水已经蒸发得差不多了——时间段比昨天在这个位置时晚了约十五到二十分钟。绕行使他的到达时间推迟了。

墙外是铁轨残余的延伸方向。轨道上方的晨光在这个时间已经消除了低角度的阴影——铁轨在砖石碎块之间反射的光线偏冷。

他在墙根下沿继续移动了约二十米。在下一个角落——一个废料堆的阴影中——停下来。

-----

口袋里。内袋。手指在接触到金属片的边缘时感知到一个变化。

金属片的温度分布与一星期前不同。他上次认真检测是在第十三天——银币测试后那次。当时手掌覆盖表面时感知的温度梯度是稳定过渡的，从边缘到中心的热辐射梯度变化平滑。今天是另一种分布——温度在表面的分布有一个明显的集中节点，在偏上的区域，约在金属片纵向三分之二的位置。

不是均匀分布平滑过渡了。局部有一个温热聚集的区域——不是外部加温的痕迹，是金属片自身在某个方向上产生的温度分布偏移。核聚。

他把金属片握在手掌中，覆盖表面。闭眼感知。

梯度方向——从底端到顶端的温度走向在一个斜面上倾斜。之前的分布是底端比顶端略低约半度，过渡均匀。今天底端到顶端之间出现了一个梯度折点——约在金属片纵向三分之二位置——温度在那个折点上升后不再继续下降，维持在一个平台期。

折点存在。他在手掌中闭眼确认了两次。第一次感知——折点在偏上位置。第二次确认——折点与金属片的宽度方向界面纵向偏移吻合。

外部环境变化后它的状态也变了。军列触发后、战争线进入地面可感阶段后、灵能背景场发生了偏移——金属片的被动响应模式出现了可识别的变化。不是他激活了什么——是外部环境的变化在金属片上留下了可检测的痕迹。

它和外面发生的事情有关系。

他把金属片放回内袋。

没有来源揭示、没有功能揭示、没有灵能关联方式的结论。只有一个确认: 它变了。在看不见的外部环境中他无法直接感知的某种场发生了变化——金属片是那个变化的指示器。他不需要理解场的性质来使用这个信息。只需要确认金属片的状态在变化后就不同了。

-----

他从废料堆的阴影中站起来。站起来时他回头看了一眼来路——宽路的路面上没有新的移动迹象。过道入口的方向没有声音。窄巷封锁的那个入口——他停住后退的那个位置——在这个角度被建筑遮挡看不见。

他沿着墙根向西偏南方向继续移动了约五十米。在下一个拐角，一条通往更开阔区域的路径入口处，他看到了今天的第一组巡逻。

两个人。从开阔区域穿过——非军人装束。但步伐的节奏和视线方向显示他们在执行巡逻任务。不像军列上的那种军事人员——可能是保安或检查站的配置人员在执行新频率的巡查。他们的视线覆盖范围覆盖了开阔区域中这段墙根出口的方向。

他在拐角处停下。等。

等待持续了约三十秒。三十秒内他保持静止，没有发出任何声音。身体在这段等待中显示了回应暴露风险时的状态——呼吸被抑制、颈部肌肉群的张力增加、眼球的扫频频率降低到仅追踪视线中边缘移动的频率。

巡逻通过。两个人消失在开阔区域的另一侧。

他走出墙根。走到开阔区域边缘时，他看到了今天不能再往前的标志——不是路障、不是岗哨——是在墙根出口外约十五米处的一段新标记。地面上喷了两条平行的白线，从路中间一直延伸到两侧建筑的基础——像是最近两三天内画上去的。喷漆的气味还残留。

白线标记的范围覆盖了通往工业带纵深方向的主要通道入口。

他站在白线前。

不是不能通过——是通行需要身份验证、登记、或某种通行程序的标志建立。白线意味着走进去不再是无须解释的移动。进入那个范围需要被看到、被记下、被询问。

他不能走更远了。

出口前方的开阔区域连接着工业带纵深——包括军列驶出的方向、昨天在装卸平台上观察到的西偏南方向、以及今天绕行后确认的多条路线交汇点。所有方向的入口都被白线标记覆盖了至少一条路径。

可走的路比昨天少了一条。不是一条——是在这个方向上所有深入工业带的路都被白线覆盖了。绕行到其他方向需要穿过更开阔的区域——暴露时间更长。今天的最远距离比昨天少了大约三百米。

他在白线前站了片刻。没有跨过去。没有测试白线的性质。他转身走回墙角时步伐的节奏没有变化——和来的时候一样的步频，方向反了。

往回走。沿着墙角加速。过道入口方向不变——走回去。

-----

回程的窄巷网络在半小时后进入了他熟悉的区域。过道→宽路边缘→第二个窄巷交叉口→装卸场北侧边缘。路线在调整了三处绕行选择后回到了第五天的起始方向。但那间今天第一个被封的窄巷入口——他没有再走回去。

他进入仓库的路径比今早离开时更短。不是说距离——是到达门廊前他减少了两次检查(一处墙角的视觉确认跳过、一条可选的窄巷不去看入口)，直接走最短的回程。

门廊。推门。

铁梯。第三阶——泛音频率与今早出门时不同。今早是踩上去的重心向前推力的泛音，现在是身体重心向后放的步伐泛音。向回走时的身体姿态与出发时不同。出发时重心在前、推向门外的方向；回程时重心偏中后、归位。

第五阶——手机操作后夹在口袋中升温的余热在胸口附近的位置可感知。经过五阶的抬升和呼吸节奏从走路模式换回室内模式的变化。

门在身后关上。

-----

透气孔中的光柱截面在地面的落点比今早出门时向右偏移了约一掌。绕行使他的到达时间推迟了约二十分钟——光柱在这个时间已经偏过了顶点，开始向另一侧回移。

灰尘在光柱中沉降的路径在特定高度形成一个可见的光锥边界——光从进入口平面到地面的过渡中灰尘密度在某个高度上发生了分级，在光柱中形成一条横向的光影分层线。他把手伸进光柱——手掌穿过分层线的位置时皮肤感知到一个微幅的温度跃变。分层线不仅是视觉——光线在穿过不同密度的空气层时的温度差异在皮肤上有一个可感知的跃迁。温度差约在零点几度到一度左右——皮肤在从光锥上段进入下段时的感觉有一个约半秒的切换。

墙角。陶罐。水面在罐口下方约三指半宽。今天消耗的水比昨天多——约三口(晨醒一口+过道中一口+回程前一口外加一处隐蔽等待中的小口)。消耗速率在冲突扩散日的压力下比日常上移了约30%。

手机。坐下时从口袋中取出。背壳温度比今早出门前更高的温度起点——但在回程途中操作和走路交替中进入了一个新的热均衡范围。对比昨天同时刻——手机背壳温度比昨天此时高约一度。体温基线在触发段跃升后的第二天没有回落——在第二个持续压迫日中维持在高位。

他坐在墙边。坐姿比今早出门时更靠前——背靠墙的角度比前几天向前了约一掌半。重心在坐骨上的位置偏右——与今早踩铁梯时的前脚掌偏右对称。身体在一天的偏移绕行和封锁感知后，把坐姿的主体重心从靠后的休息态重新校准到了更接近站立态的前倾位置。

光柱中的灰尘在维持缓慢的沉降循环。分层线在他收回手后恢复了完整的视觉边界。

今天比昨天少走了约三百米。比昨天更早退回了仓库。比昨天多绕了一处窄巷。今天白线标记出现在地面，巡逻的频次升级，金属片的状态发生了变化。

每天可走的范围在收缩。不是急剧的——是每天醒来发现比昨天少了一条路、多了一条限制、多了一项需要重新评估的日常判断。积累的方向是从大到小、从多到少、从自由到有条件。他在地下室坐下的位置——近六天来每日向门口偏半掌到一掌——已经离墙壁的距离增加了约一掌半。身体已经在为明天的更早出门、更快返回做准备。

光柱在地面上的落点在他坐下的这段时间里继续偏移了约半指宽度。水面上的椭圆亮斑在壁面反射的光线变化中保持着稳定的形状。灰尘在光锥中继续沉降，手机在口袋中冷却。

门框在今天的积累方向是向右偏——和上一周期的初始方向一致。第二天的热累积已经在可检测范围内。明天的第三天的增量将决定这个周期是否在方向上完全重复上一周期，还是在积累过程中开始偏移。

他用手指在身边的墙面上摸了一下指尖的触感。灰。他在这块墙上靠了六天，灰层已经被衣服反复擦过——在指尖接触到墙面的位置有一层微滑的质地。是衣服纤维在灰层上摩擦形成的光滑化。这是他在这个位置坐了六天的痕迹。

水在陶罐中随着他坐下的动作微晃了一下。水面上的光线反射在晃动中散成细小的碎斑——在罐壁上闪烁了约一秒后归位。

明天可走的路会比今天更少一条——还是不止一条。
