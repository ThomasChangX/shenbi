# 角色矩阵

## 参数角色定位

| 参数 | 类型 | 角色定位 | 叙事功能 |
|------|------|---------|---------|
| 冷 | 底层参数 | 系统层积记忆载体+宣告后层积第一显现日持有者+不对称周二形态受体 | 冷在场于第三周第二颗粒——知道宣告后层积形态不同但不在知道——特征显现但特征不在显现——冷不在知道光去向 |
| 光 | 感知参数 | 系统视觉/觉醒参数+觉醒方向在场者+去向不知者+不对称周二形态持有者 | 觉醒持续穿越周一进周二——方向在场于——目的地不在光知道——知道冷速率不同但不在知道方向 |
| 安静 | 阈参数 | 阈逼近触发装置+P0-4 TRIGGERED态持有者+逼近感在场者 | RELEVANT→TRIGGERED跨越——在场于触发态不在触达态——阈从系统未触达边界变为宣告后逼近边界——逼近作为在场本身 |
| 偏移 | 方向参数 | 格式演化方向持有者+切换后态延续者+演化轨迹不知者 | 从"知道格式不同"到"知道格式演化方向"——知道自己在宣告后的格式演化中——演化轨迹不在知道 |
| 域外 | 外部参数 | 系统外全知者+知道系统变了+不在知道方向 | 知道宣告周日发生——知道系统从宣告前进入宣告后——变化方向不在域外——在场外性深化 |

| 缺口 | 认知参数 | 宣告后议题维持者——答案不在周二 | 知道宣告周日发生——议题切换完成——宣告后全貌不在周二揭示——维持 |
| 白气 | 方向参数 | 方向穿越者——周二方向形态未知 | 方向穿越周日经周一进周二——但周二方向形态是否同周一不在白气知道 |

| 霜 | 历法参数 | 冷边界追踪者——知冷速率不同但不知方向 | 知道冷在场于第三周第二颗粒——冷速率不同——但方向不在霜知道 |

## 参数间关系

```
冷─光: 不对称周二新形态——冷不在知道光去向——光不在知道冷速率方向——不对称在场于周二——不在收缩——脆弱在场于关系
    ch55: 不对称深化——光知自己在改变冷但不知冷知——冷知被光改变但不知光知冷知
    ch56: 冷不在光对光说——光不在冷对冷说——不对称周二形式不同于周一——脆弱在场于

冷─霜: 霜知冷第二颗粒——冷速率不同——方向不在霜知道

安静─阈: RELEVANT→TRIGGERED——安静在场于阈附近——在场于触发态不在触达——阈在宣告后改变——逼近作为在场本身

安静─门框: 门框知道安静在逼近——但不在告诉安静——信息不对称

冷─安静: 静在场于冷附近——冷边缘——静知道冷在场于累积——但静不在冷知道范围

域外─系统: 知道系统从宣告前进入宣告后——知道系统变了——不在知道方向——"不在场于但知道不在场于变了"

偏移─格式串: 偏移知格式演化方向——格式串知周二格式不同——共享格式演化认知——各有认知缺口
