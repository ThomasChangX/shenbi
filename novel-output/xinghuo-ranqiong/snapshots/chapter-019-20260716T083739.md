# 绝

透气孔中的光柱在墙面上留下的落点比前两日此时更接近墙角——第三天。光线方向在第三天不需要睁开眼就可以通过皮肤上的热辐射变化判断时间段的偏移。身体学会了在亮度到达之前通过室内空气温差的日间倾斜来锚定时刻。

睁开眼睛时眼睑的运动轨迹与前两日不同——今天睁开眼睛的角速度比昨天快。不是清醒程度的差异——是身体在睡眠中完成了更高的恢复效率后切换姿态的速度在提升。适应过程在加快。不是好或坏——是身体在持续暴露中减少了在每个阶段停留的时间。

坐起来。手指触及手机——背壳温度比前两日同时刻的基线进一步上移。约零点五度的增量。体温恢复基线在第三天没有回落的迹象，进入了新的稳态平台。触发段跃升的第三天——身体在新的基线位置上运行。

蹲下。门框前。

闭眼。手指向前延伸——在接触到木框表面的前一刻，他已经知道左边缘的下方将有偏移。热辐射的方向——偏右。和前两日相同的侧向。方向可重复性在第三天确认。

感知从手指向木框表面延伸。左边缘偏下——温度梯度分布线在木框表面的倾斜方向与第二日一致。偏移量——约比第二日此时的感知增量小了一档。第一天到第二日的增量大(白纸起点→可见单侧累积)，第二日到第三日的增量在收缩。比例约占上周期同段位的65%至70%。

但有一个差异。

手指在垂直方向的主梯度之外感知到一个表层差异——约在木框边缘与内侧面交界的纵向上。不是主梯度的延续。是在主梯度之外存在一个可独立分辨的次级分布——方向偏右偏上，幅度比主梯度小约两个可分辨层级。

横向偏移。

上一周期的第三天结束时没有出现过这个方向上的单一可分辨层。

他在木框前保持闭眼姿态——感知在这个横向差异上停了一拍。不是认知判断——是感知系统在识别到一个新增差异维度时需要的验证时长。手指从感知表面收回前，在木质纹理的同一纵向位置重新确认了一次。梯度指向的偏上偏右角度在第二次触摸中保持在同一个轨道上。

横向不存在于上一周期的同阶段。新周期在重复中产生了自己的变形。周期的方向模板不是机械复制——每次的底色环境参数不同——在方向匹配后出现了上周期没有的次级维度。

他睁开眼。站起来。推门。

第一个方向——右转。

-----

仓库外墙与第一个转角之间的空气流动在门廊外传递了一个可辨识的化学信号。喷漆的气味。不是前天在工业带入口处的残留——是更新的操作。溶剂挥发状态的阶段性——从气味的刺激强度判断，施工时间约在六到八小时前。距离比前天更近。扩散的浓度梯度在转角第一个方位时达到可分辨阈值。

他停住。

停住不是因为气味有危险——是在他的路线网络地图中，气味来源与窄巷中段的位置关系发生了一次重映射。气味来自他的日常路线方向——窄巷中段。不是来自前天白线标记的方向。

他朝气味走了一段。转过第一个拐角。

围挡在窄巷中段横贯两侧墙体之间——从地面延伸到约两米半高度。铁丝网在钢结构框架上绷紧的间距均匀——不是临时放置的应急围挡。框架嵌入墙体——两侧延伸的金属支架打入砖缝后在水泥中固定。不是今天下午拆、明天再架的设施。

喷漆的气味从这里来。围挡的金属表面喷涂的防锈漆。溶剂在六到八小时前挥发到可扩散至仓库外墙的浓度。施工在夜间进行。他睡着的时候有人在距他不到二百米的位置立了这堵墙。

窄巷中段——他在第三天走到的位置。中断。

前面两天他从这条窄巷走。今天不行了。围挡不留间隙。墙两侧没有可以绕过或钻过的余地——框架与墙体的接缝处不留超过一掌宽的缝隙。

他站在围挡前。停的位置比围挡表面远约一臂——保持距离。不是出于安全考虑——是身体在看到围挡的结构性嵌入之后自动停在了一个进可观察、退可绕行的中立位置。

铁丝网在晨光中的阴影在地面上拉出网格状的投影。网格密度在墙体的上方形成一个缺口——围挡上方留出的缝隙透入一线光。不足以翻越。翻越的代价——高度、倒刺、落地声响——在那个时间段已经计算过了。没有翻越的价值。

他转身。回去走另一条。

-----

绕行路线偏西。他在一个废弃装卸台的西侧沿墙根向北偏西方向移动时——前方约十五到二十米外的转角处传来人声。不止一个人。对话的语调在墙与墙之间反射——音节内容无法辨认，但声调的角度有不耐烦的比例。

他贴墙。放慢。贴近转角的墙壁——在转角内侧停在距离转角约两臂的位置。

喷漆的气味。从转角外传来——不是围挡的气味，是另一种。沿着地面扩散的浓度——是一种更稀的、来自地面的新涂层。和前天的白线相同的化学成分——干燥进度推算出更短的时间差。约二十四到三十六小时前施工。

转角外——一条通向工业带纵深的主通道入口。地面上的两道白线。和前天工业带入口相同的位置模式——平行线从路面延伸至两侧建筑基础。

但今天白线内侧有人。

一个穿灰色外衣的人站在白线前——外侧——面对两个站在白线内侧的深色外套。灰衣服的人面向两个深色外套，身位没有越线。一个深色外套的人在开口前视线从灰衣服的人的脸上移到他的手上、再移到口袋位置——视线覆盖的轨迹是标准的身份验证前的检查扫描。灰衣服的人的手在口袋位置停了一下——摸了两侧口袋——没有取出东西。

深色外套摇了摇头。幅度不大。

灰衣服的人站在原地站了约两秒。然后转身。沿来路返回。经过转角时与林烽的间距约五米——灰衣服的人没有看他。视线向下——在返回的状态中。

林烽在转角后保持贴墙姿态。

白线不是线。白线是有人站在线的内侧检查通过的人——你没有身份证明，就站在外面。

他在转角后没有发出声音。灰衣服的人走远后他从墙根走回。回程的路线不需要经过白线的视野范围——他在进入第二个转角前沿墙体偏东的方向移动了约十米，然后转入一条夹缝。

-----

夹缝——两侧都是高墙的窄通道——通向一个没有门板、没有窗的废弃空间。进深约三米，宽度不足两臂。顶部的缺口透入一线天光，不足以照亮空间内的物体轮廓——仅在地面上形成一个约手掌宽的亮条。除亮条外的区域全暗。

他走进去。完全封闭空间。

取出MH-020。屏幕亮起时的光在这个空间的暗背景中形成一个可见的反射——墙壁上的涂料层在屏幕光的散射中显示表面覆盖着沉积多年的灰尘。

折线图标的位置——在屏幕的左下区域。展开稳定。波形在这个空间中没有大幅度的随机横移，没有跳变。尾缀: [s]。

数值: 十二点零。十一点九。十二点一。

波动幅度约±0.1——比昨天在半开敞过道中的读数波动(约±0.4)小了一个数量级。尾缀[s]在这条完全封闭空间中保持稳定。没有跳变到[n]或[d]——位置的物理开敞度在这个空间中已经降至最低值，尾缀在零变动状态下输出[s]。

他把手机持握方向旋了九十度——垂直方向。竖立标尺在屏幕上的位置——折线图标在姿态变化后没有跟随重力方向明显偏移。不是方向敏感。他移动——从空间的左侧墙面换到右侧墙面。折线图标在横向坐标上偏移了约三像素——左墙边的位置偏左，右墙边的位置回到屏幕中心。

竖立标尺在不同位置输出有差异。不是方向——是位置。

他蹲在这个位置的时间——从取出到完成两次位置换位——约六到七次呼吸。操作时长。折线图标在第六次呼吸左右的时刻从稳定切换——开始向屏幕的右侧缓慢横移。不是位置变化引起的——他从左墙边回到右墙边后没有移动过。是操作时间在某个阈值时触发了显示模式的切换。

第三维度——操作时间。超过特定持续时间的操作激活了一个附加信息的接入——横移速率告诉他这个空间的参数在长时间持续观察中不是表面呈现的稳定，而是在持续暴露中向某个方向累积变形。

他收起手机。收起动作的速度——比前日快。

-----

从完全封闭空间退出。沿夹缝返回主通道。

在距离主通道出口约五米的位置——他听到了脚步声。方向——主通道入口。林烽停住。声音——至少两个人的拼接脚步声。间距不是一人走——是两人并行，步伐不完全重合。

他退了两步。

夹缝没有岔路。没有侧门。没有可以遮蔽的分支空间。长条形——进深约十米，宽度仅供一人通过。他在距离出口约七米的位置。脚步声在入口方向——与直线出口之间没有遮挡。如果他处在入口方向的人的视线中的话——长条形通道在光线状态下至少在前段轮廓可辨。

他没有继续退。在七米位置停下。

背靠墙。呼吸抑制至最低维持深度。颈部的张力将头锁定在位置——使视线在黑暗中以最低频率扫动覆盖通道入口方向的边缘。

脚步声——在入口位置停了一下。有人在夹缝入口的方向停留——不是走进去，是在入口外侧停住，看什么或等什么。

林烽在夹缝内没有移动。约三四次呼吸的停顿——脚步声重新启动。没有拐进来——沿着主通道方向——向外侧——远了。

他在墙边保持静止。呼吸从抑制状态释放——但底层的唤醒水平没有降回到进入前的基线。

口袋中的金属片。

手指在从墙边移开的动作中接触到金属片边缘时——脉冲。一次单独的振动——从金属片的表面经过手指传到指腹。波形——单次——不是连续震荡——幅度约等于一瞬的肌肉收缩。但他的前臂在这个瞬间没有收缩。他在触摸到金属片时手指是静止的。脉冲在接触前已发生。

不是他的手在抖。

他把金属片取出。手掌展开。金属片在手掌上——没有持续振动。刚才的脉冲已消失。没有后续。单发的。

他握紧。

在封闭的夹缝中——退不出的空间——前方向主通道的方向——脚步声可能折返。在无法退、无法进的节点——金属片给出了一次响应。不是被动的温度变化——是主动的、瞬时的、非连续的。

他在夹缝中保持站立姿态。握着金属片。没有再出现振动。

外面没有声音了。

他收起。放回内袋。

它动了一下。他不知道它为什么动。不知道它什么时候会再动。但他知道它能动——在他没有碰它的时候。

-----

回程。路线偏北。

窄巷中段的围挡不可通行。白线执行域在主通道入口占据了一条主要出口。巡逻频次的变化——今天他在移动中没有一次遇到超过十五秒以上的安静间隔。每条他在前两日形成的自发间隙都在今天被压缩或消失。

他在第三次绕行时进入了一个三岔方向——其中一条是今早来时的路。但那条路在白线的延伸方向上。走回去意味着在上午的日光中穿过标记覆盖域的视线范围。他没有走——选了更北的方向。比来时多绕约三百米。

到达仓库时天色尚在偏午之后——光柱已经从垂直位置偏向了东侧。回程时间比前两日早——不是回程更短，是他在白线场景后就决定了回程。有效活动时间比前一天短了约四十分钟。

门廊。推门。

铁梯。第三阶——泛音的频率与今早出门时不同。今早的重心在蹬踏时向前——身体推着向门外的方向。回程时重心偏后。泛音在第三阶的释放角度向后——归位而不是推进。

第五阶——口袋中的金属片的位置在体内感知频率的映射中与出门时不同。不是口袋不同——是他知道它在他不知情的情况下有能力主动响应。那个脉冲改变了他在口袋中对它的体感映射。不是在口袋里放了一个"测量设备"——是在口袋里放了一个"会在你不知道的时候动的东西"。

地下室。透气孔光柱落点在上午出发时位置的右侧。

墙角。坐下。水面在罐口下方约二指半——消耗比昨天同时间高。今天的活动时长更短但代谢成本更高。

他从内袋取出金属片。

手掌展开。金属片在掌中——静止。他把金属片放在地面瓷砖上——手离开。金属片在地面上——没有任何运动。晨光中它的形状和角度没有变化。

他等了约五次呼吸。没有。

拾起。放回。

它动了一次。他没有办法预测下一次。

-----

光柱在地面上的落点在他坐下的这段时间里继续偏移了约半指宽度。灰尘在光锥中沉降——在分层线上下循环。水面上的椭圆斑在罐壁的光反射中保持稳定的形状。

身体靠墙的角度比出门前更靠后——但比同时间标准休息态偏前约一掌。一天的感知负荷在坐下后释放了部分张力——但底层的唤醒水平没有回到出发前的基线。今天的白线执行、围挡封锁、夹缝等待和金属片的脉冲——这些事件在身体上留下了可测量的痕迹——心率在安静状态下的基础节奏比同时间的昨天快了约三到五个基点。

门框——第三天的积累产生了与上一周期65-70%一致的偏移方向——但出现了横向偏向上方的次级分布。上一周期第三天时不存在的维度。周期开始产生自己的变形。

手指在身边的墙面上——他在这个墙角坐的位置在七天内已经积累了可触摸的痕迹——灰层被衣物反复擦拭后在指尖下形成了光滑与粗糙交替的轨迹。
