- **核心偏离**: 林烽×阿莲 — 从"拯救→短暂相处→死亡（原作）"变为"拯救→共同成长→CP发展（CP模式）"

## 内容

### 核心CP关系: 林烽 × 阿莲

**原作关系轨迹**:
```
Ch2: 林烽在垃圾场发现并救起被权贵抛弃的阿莲
Ch2-3: 短暂相处
Ch3: 阿莲在空袭中死亡——林烽悲愤觉醒
后续: 阿莲作为"已死之人"成为林烽记忆中的道德驱动力之一
```
- 引用: [大纲] outline-example.md Ch2: "在垃圾场发现一个被权贵玩弄后抛弃、奄奄一息的底层少女"
- 引用: [大纲] outline-example.md Ch3: "那个被他救下的少女，在火光中化为灰烬"

**CP模式关系轨迹（偏离后）**:
```
Ch2: 林烽在垃圾场发现并救起阿莲 [保留]
Ch2-3: 短暂相处中建立初步信任 [保留+扩展]
Ch3: 空袭中林烽保护阿莲——灵能感知首次主动使用（保护欲驱动）
Ch3: 阿满之死触发林烽觉醒，阿莲存活 [偏离核心点]
Ch4-6: 阿莲在组织中成长，与林烽从拯救者-被救者关系逐渐转为并肩关系 [新增]
Ch7: 田国栋牺牲后，阿莲成为林烽的情感支撑 [新增CP互动]
Ch8-9: 最低谷时的互相扶持——阿莲的底层出身让她与矿工天然亲近 [新增CP互动]
Ch10: 叛徒出卖——可能共同被捕或阿莲在外营救 [CP危机点]
Ch11-12: 越狱/重逢——CP关系的考验与深化 [CP高潮]
Ch13-15: 并肩作战到最终——从"两个个体"到"一对伴侣+两个战士" [CP结局]
```

**CP关系特征（偏离后的正典定义）**:
- 关系基础: 拯救者的责任感 + 被救者的信任 → 转型为平等的并肩战友
- 权力动态变化: 前期林烽为保护者（权力不对等）→ 后期阿莲通过自身成长实现关系平等化
- 情感维度: 在革命战友情中自然生长出的伴侣情感——不突兀，不喧宾夺主
- CP定位: CP是情感副线，不取代主线（林烽的觉醒与革命），不稀释政治主题
