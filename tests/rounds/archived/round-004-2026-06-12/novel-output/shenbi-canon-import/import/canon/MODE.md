# 模式声明

**选定模式**: CP（Character Pairing / 角色配对模式）

## 模式说明

CP模式是用于同人创作的四种fanfic模式之一。在该模式下，原作的关系正典被保留作为基础框架，但特定配对/CP关系被改变——例如改变相遇时间、相遇场景、初始关系状态等。CP模式的核心操作是：保留角色个体性格正典，但重新定义他们之间的互动关系。

适用场景（基于测试scenario）: 基于50章原作《星火燃穹》创作CP向同人。目标配对未在scenario中指定——本测试默认以"林烽 × 阿莲"作为CP目标（原作中阿莲在第2章被救、第3章死亡——CP模式将其关系线从"拯救→死亡"改为"拯救→共同成长→CP发展"）。

**CP模式规则**:
1. 角色个体正典保留（性格、声音指纹、弧线基础）
2. 关系正典被重新定义（配对关系发生变化）
3. 事件正典中与CP相关的关键节点被调整
4. 其他正典（世界、时间线）基本保留

## 保留清单

- 世界正典（SECTION 1）: 全部保留——灵能规则、种姓制度、势力格局、历史纵深
- 角色正典（SECTION 2）: 个体正典保留——林烽性格、阿莲身份、田国栋等
- 事件正典（SECTION 3）: 第2-3章事件线被大幅偏离（见偏离清单），其他事件基本保留
- 关系正典（SECTION 4）: CP配对关系被重新定义（见偏离清单）
- 时间线正典（SECTION 5）: 基本保留——Ch3时间锚点被调整为"阿莲存活线"

## 偏离清单

- 偏离1: 阿莲在第3章空袭中存活（原作中死亡于空袭） → CP模式核心偏离——阿莲存活并与林烽共同度过后续章节
- 偏离2: 林烽与阿莲的关系从"拯救→死亡→悲愤觉醒动力"变为"拯救→共同成长→相互支撑→CP发展" → CP配对关系建立
- 偏离3: 第3章的道德觉醒触发事件从"阿满之死+阿莲之死"变为"阿满之死"为主，阿莲的存活软化了林烽的情感冲击但深化了他的保护欲
- 偏离4: 第7章田国栋牺牲后，阿莲成为林烽最重要的情感支撑——原作中林烽独自承受
- 偏离5: 第10章叛徒出卖中，阿莲可能面临危机或被俘——增加CP向的"营救"情节动机
- 偏离6: 林烽的情感弧线从"孤独的革命家"增加"并肩的情感牵绊"维度

## 不混用

- 本作品不与其他模式混用（非AU，非OOC，纯CP模式）。角色性格不作偏离（非OOC），核心设定不作变更（非AU），仅调整角色间配对关系的轨迹。
