# 信息边界矩阵：星火燃穹

> 记录每对角色之间的信息对称性。信息差是戏剧张力的核心来源，每一处 ASYMMETRIC 都是一个可兑现的伏笔。

## 信息状态定义

| 状态 | 含义 |
|------|------|
| SYMMETRIC | 双方信息相同 |
| ASYMMETRIC | 一方知道而另一方不知道（记录谁多知道什么） |
| ISOLATED | 完全无信息交流 |
| MUTUAL_SECRET | 双方共享一个第三方不知道的秘密 |

## 信息差驱动的戏剧张力

### 已锁定 (可在当前大纲中兑现)

1. **林烽不知列塔尼亚的存在（第1-2章）** → 第3章空袭炸弹商标为首次暗示 → 第二幕中期确认真相
2. **林烽不知前穿越者历史（第1-2章）** → 第2章铭文金属片首次物质接触 → 第二幕老政委揭示历史
3. **林烽不知买办叛变（第一幕）** → 第二幕后期买办公开反叛 → 根据地遭毁灭性打击
4. **少女知道权贵内幕但未传达（第2-3章）** → 永久信息裂痕 → 林烽可能后期从其他渠道补全
5. **涅普敦不知林烽为变量（全文前半）** → 林烽利用此信息不对称实施运动战 → 第二幕中点"虚假胜利"时暴露

### 待开发 (可在后续大纲中展开)

6. **老政委的真实历史身份** → 可开发为个人前史暗线，对其与遗存的互动产生深层影响
7. **列塔尼亚对林烽从无视到重视** → 第三幕列塔尼亚开始针对性行动的反转张力
8. **权贵中观望派的信息缺失** → 可发展为潜在转化对象——当他们了解列塔尼亚的真实意图后可能转向
