# 章节模式分析

对已生成章节进行模式分析，识别重复结构、节奏问题和可用模板。

## 检测到的模式
- 章节开头模式：每章以具体场景切入，不使用概述式开头。有效。
- 段落节奏模式：平均段落长度在50-150字之间波动，符合网文阅读习惯。
- 对话/叙述比例：当前对话占比约35%，建议增加到40-45%以增强节奏感。

## 模板建议
基于检测到的有效模式，建议建立以下写作用模板：
1. 开局切入模板：具体动作/感官细节→环境白描→内心反应
2. 冲突升级模板：外部压力→内部反应→决策→行动
3. 章末钩子模板：信息不对称暗示→未解决的张力→直接提问
