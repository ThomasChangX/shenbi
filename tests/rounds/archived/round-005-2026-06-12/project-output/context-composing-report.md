## P5 世界铁律

**来源**: `world/rules.md`

1. 铁律一：灵能守恒律——任何灵能使用必须有明确的能量来源和转化步骤
2. 铁律二：知识驱动律——林烽对净化和清理的灵能运用必须建立在他理解相关物质结构的基础上
3. 铁律四：种姓社会铁律——据点建设必须避开明耀者家族的势力范围
4. 铁律七：思想转变需过程律——林烽的利己到觉悟转变必须在第10章中表现为"我愿意帮你，但不代表我加入你们"
5. 铁律八：灵能科技需基础律——林烽的设施建设不能超越他本科通识水平
