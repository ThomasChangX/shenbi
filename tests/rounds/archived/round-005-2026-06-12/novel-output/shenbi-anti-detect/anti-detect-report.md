# 反检测改写报告

## 改写报告

**章节**: 第1章（样章）
**触发原因**: 初始审计发现 AI 标记

### 应用手法

| 手法 | 改写位置 | 原文标记 |
|------|---------|---------|
| 打破句式规律 | 第2段 | 连续3句相同结构 |
| 了字降频 | 第4段 | 4个"了" → 2个 |
| 转折词降频 | 第3段 | "然而""但是" → 场景转换 |
| 情绪外化 | 第5段 | "感到愤怒" → 手掌掐进掌心 |
| 感官锚定 | 第1段 | 抽象"破败" → 铁锈味+废渣硌脚+远炉轰鸣 |
| 对话差异化 | 第6段 | 催收员加粗鲁口气，林烽保自嘲幽默 |
| 打破段落等长 | 全文 | 均衡4段 → 长短交错 |
| 替换分析术语 | 第5段 | "由此可见" → 心里就一个想法 |
