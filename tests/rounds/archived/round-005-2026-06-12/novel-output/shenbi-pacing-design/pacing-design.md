## 节奏设计报告

**项目**: 星火燃穹：位面解放战争史
**设计时间**: 2026-06-12

### 节奏周期

每周期8-15章，包含四拍：

| 拍 | 功能 | 占比 |
|----|------|------|
| buildup | 积累张力 | ~30% |
| escalation | 升级冲突 | ~25% |
| explosion | 高潮爆发 | ~20% |
| aftermath | 沉淀消化 | ~25% |

### 三线比例

| 类型 | 定义 | 目标比例 |
|------|------|---------|
| QUEST | 引入新信息/角色/地点 | ~20% |
| FIRE | 高强度冲突/转折/决战 | ~60% |
| CONSTELLATION | 沉淀/关系/日常 | ~20% |

### 场景类型（7种）

| 类型 | 检测标准 |
|------|---------|
| 战斗 | ≥2角色物理冲突，含攻防描写 |
| 对话 | 对白占比>50%的章节 |
| 探索 | 进入新地点，发现新规则 |
| 日常 | 低冲突，关系推进为主 |
| 训练 | 能力提升，技能学习 |
| 阴谋 | 计谋推进，政治博弈 |
| 回忆 | 闪回过去事件 |

### 连续限制

同类型最多连续3章。检测方法：逐章标注类型，滑窗检查。
