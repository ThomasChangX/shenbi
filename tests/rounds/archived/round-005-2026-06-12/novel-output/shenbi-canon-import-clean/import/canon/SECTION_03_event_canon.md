# SECTION 3: Event Canon（事件正典） — CP模式

**正典模式**: cp
**源**: 《星火燃穹》/ outline-example.md + short-story-map-example.md + chapter-draft-example.md
**导入时间**: 2026-06-12

## 保留范围

事件正典的主体结构保留——三幕框架、关键转折点、章节级事件线。CP模式仅调整与CP配对直接相关的第2-3章事件节点。

## 偏离声明

- 关键偏离1: **阿莲存活于第3章空袭** —— 原作中阿莲在Ch3空袭中死亡，CP模式中她存活
- 关键偏离2: **第3章林烽的觉醒触发事件调整** —— 原作由"阿满之死+阿莲之死"双重触发；CP模式由"阿满之死"主触发，"阿莲差点死去"作为强化触发
- 关键偏离3: **后续章节增加CP互动节点** —— 第4-15章中增加林烽与阿莲的互动场景（并肩作战、情感支撑、危机与营救等）
- 非偏离: 主线战役、田国栋牺牲、叛徒出卖、狱中突破、首战告捷等核心事件全部保留

| 章节 | 原作事件 | CP模式调整 |
|------|---------|-----------|
| 4 | 帮田国栋重整民兵，师徒建立 | 保留。**[新增CP]**: 阿莲开始学习基础灵能/组织知识，与林烽并肩 |

| 7 | 田国栋牺牲，林烽被迫领导 | 保留。**[CP强化]**: 阿莲成为林烽在田国栋死后的情感支撑——非替代老政委，而是新增的牵绊 |

### 关键转折点（CP调整后）

1. **第3章**: 陈阿满之死 → 林烽觉醒（保留）。**[CP]** 阿莲存活 → 觉醒从"纯悲愤"变为"悲愤+保护"
2. **第7章**: 田国栋牺牲 → 林烽被迫领导（保留）。**[CP]** 阿莲的情感支撑——林烽不再完全孤独
3. **第10章**: 叛徒出卖+被捕（保留）。**[CP]** CP危机——分离或共囚
4. **第12章**: 灵能突破+联合（保留）
5. **第14章**: 胜利+代价（保留）。**[CP]** 双人面对胜利——情感重量增加

### 时间线（Ch1-3关键节点）

```
Ch1 (~12h): 穿越 → 探索 → 催收 → 独处（截止期限: 明日日落）

Ch3 (CP偏离): 阿满处决 → 空袭 → 阿莲【存活】→ 林烽觉醒（保留）
```
