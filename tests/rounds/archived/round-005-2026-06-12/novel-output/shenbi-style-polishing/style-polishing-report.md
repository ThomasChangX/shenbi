# 风格润色报告

## 润色说明

## 润色目标
基于style/style_profile.md指纹进行靶向润色。

### 润色维度
| 维度 | 目标 |
|------|------|
| 句长节奏 | 中短句>70%, 长句仅反思/设定 |
| 段落呼吸 | 行动段2-3句, 描写段4-6句, 无连续等长 |
| 幽默注入 | 每2-3章1次林烽自嘲或孟泽枯涩幽默 |
| 感官优先 | 每章>=1触觉或嗅觉描写 |
| 对话比例 | 20-25%, <=30% |

### 润色示例
原文: "林烽感到愤怒和无力，他意识到自己在这个世界的处境比他想象的要糟糕得多。"
润色: "林烽的手掌掐进掌心。疼。行吧，这个世界就不是给底层留路的。他把这个想法咽回去，换成一个自嘲的笑。"
手法: 情绪外化 + 了字降频 + 自嘲语气

### 审计对比
| 指标 | 润色前 | 润色后 |
|------|--------|--------|
| 平均句长 | 28.3 | 24.1 |
| 连续等长段落 | 3处 | 0处 |
| 幽默密度 | 1次/4章 | 1次/2章 |
| 感叹号密度 | 4.2/千字 | 2.3/千字 |
