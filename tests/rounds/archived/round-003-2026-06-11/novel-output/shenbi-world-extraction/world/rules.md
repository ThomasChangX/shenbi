# 世界铁律（反向提取）

> 最多 10 条。每条 >= 2 个证据。
>
> **证据类型标注**:
> - 「定义型规则」= 世界观中显性陈述的规则（可引用 world docs 或章节文本）
> - 「演示型规则」= 通过角色行为/情节展示的规则（必须引用章节文本，不得仅引用 world docs）

## 规则 1: 灵能守恒定律 [定义型规则]

灵能总量在整个灵质海中恒定不变。灵能不能凭空产生，也不能凭空消灭——只能被重新分配、转化形态、或从一个载体转移到另一个载体。

- 证据: 第1章 灵能炉的低频震颤"每七八秒震颤的频率变一次，像某种正在进行中的能量循环"——暗示能量在转化而非生成（章节文本）
- 证据: 第5章 "能量不会凭空消失——他把一束灵能分解了，消耗的是自己体内的灵能储备。等价交换，一分没差。"——林烽对自身灵能消耗的体验（章节文本）
- 证据: world-story-bible.md 第一段明确陈述"灵能无法凭空产生，也无法凭空消灭"（world doc）
- 证据类型: 显性（定义型——world doc + 章节文本双重确认）

## 规则 2: 知识-灵能对应律 [定义型规则]

角色可运用的灵能上限由其知识水平严格决定。

- 证据: 第1章 林烽分解矿渣——"像一个程序在执行'拆分'指令：铁归铁，硅归硅，微量灵能结晶被剥离出来"——物质分离纯化的概念来自现代化学知识（章节文本）
- 证据: 第5章 林烽分解白光——"他能'看到'白光的内部结构：不是一束连续的能量，而是由数千个微小的灵能单元排列成网"——系统分析思维使他能理解能量结构（章节文本）
- 证据: world-story-bible.md "一个农夫可以把灵能化作锄头，但他永远无法用它打开一条位面通道，因为他的认知体系里没有'位面'这个概念"（world doc）
- 证据类型: 显性（定义型——world doc + 章节文本双重确认）

## 规则 3: 位面通道三条件 [定义型规则]

打开位面通道必须同时满足坐标计算、能量阈值、稳定方程三个条件。

- 证据: world-story-bible.md "列塔尼亚之所以能建立殖民霸权，不是因为他们的军队最多，而是因为他们垄断了位面通道的数学"（world doc）
- 证据: world-story-bible.md "打开通道所需灵能与两个位面之间的灵能密度差成正比。密度差越大，通道维护成本越高"（world doc）
- 证据: world-rules.md 规则三——坐标计算、能量阈值、稳定方程三个条件的完整定义（world doc）
- 证据类型: 显性（定义型——第1-5章未出现位面通道操作场景，规则完全来自 world docs）

## 规则 4: 种姓灵能禁锢律（梅德兰特有）[定义型规则]

梅德兰的种姓制度通过法律规定不同阶层的灵能使用权限。

- 证据: 第1章 墙上告示《梅德兰帝国灵能教育债务管理条例》——"债务人不得以任何理由离开债务登记区，违者以灵能僭越罪论处"（章节文本）
- 证据: 第1章 催收员台词——"根据《梅德兰帝国灵能教育债务管理条例》第十二条，灵能教育贷款适用特别利率标准，不受常规民间借贷利率上限约束"（章节文本）
- 证据: 第3章 田国栋讲述——"庶民不算工伤"（帝国不给矿工工伤赔偿）（章节文本）
- 证据: world-story-bible.md "底层是占人口百分之八十五的'灵能庶民'——被法律剥夺灵能教育权利的劳动者"（world doc）
- 证据类型: 显性（定义型——world doc + 章节文本双重确认）

## 规则 5: 殖民三支柱结构 [定义型规则]

列塔尼亚帝国对任何被殖民位面的控制通过技术垄断、买办经济、话语体系三个支柱运作。

- 证据: 第1章 灵能炉铭文——"列塔尼亚制造标记"（技术垄断的痕迹）（章节文本）
- 证据: 第1章 催收员两次提及"上面定的规矩"——模糊的决策层暗示殖民体系的层级控制（章节文本）
- 证据: world-story-bible.md "列塔尼亚不靠军队数量统治世界——它靠三样东西：位面通道的数学垄断、灵能理论的知识霸权、以及一套精心设计的'文明等级论'话语体系"（world doc）
- 证据类型: 显性（定义型——结构由 world doc 完整定义，章节文本提供环境层面的印证）

## 规则 6: 位面物理差异律 [定义型规则]

不同位面之间存在至少一种物理差异（时间流速、重力、灵能密度）。

- 证据: world-story-bible.md "在涅普敦位面度过一年，梅德兰位面可能已经过去了三年"（world doc）
- 证据: world-story-bible.md "涅普敦位面的重力是梅德兰的一点七倍"（world doc）
- 证据: 第1章 天棚缝隙漏进的天光"颜色偏紫，跟地球上的夕阳不太一样"——暗示所在位面大气成分或恒星光谱与地球不同（章节文本）
- 证据类型: 显性（定义型——位面级物理差异定义来自 world docs，细节印证来自章节文本）

## 规则 7: 金手指边界律 [演示型规则]

林烽的灵能天赋存在明确边界——分解、塑形、融合三类，精度受知识水平限制，不能分解有生命意识的对象。

> ⚠ 演示型规则：证据必须来自章节中角色实际使用灵能的行为，不得仅引用 world docs。

- 证据: 第1章 林烽分解矿渣——"矿渣在他手心里碎开了。不是碎裂——是分解。像一个程序在执行'拆分'指令：铁归铁，硅归硅"——展示分解能力的具体运作（章节文本·行为演示）
- 证据: 第5章 林烽分解白光——"白光的灵能结构在他的感知层里像一张网，他的灵能本能地把那张网拆散了"——分解能力应用于能量结构（非生命体），印证"不能分解有生命意识的对象"的边界（章节文本·行为演示）
- 证据: 第5章 林烽分解白光后"皮肤下的暗河干涸了，像一条被抽空的水渠"——灵能消耗的代价边界，展示"等价交换"原则（章节文本·行为演示）
- 证据类型: 演示型——所有证据均来自章节中角色实际行为

## 规则 8: 穿越者认知摩擦律 [演示型规则]

主角林烽作为现代穿越者，其行为必须持续展示与异世界的认知摩擦。

> ⚠ 演示型规则：证据必须来自章节中角色的认知行为描写，不得仅引用 world docs。

- 证据: 第1章 穿越冲击——"灵能修炼贷款，他把这几个字念了一遍，像在测试一种陌生语言的口感"（语言/概念摩擦）（章节文本·行为演示）
- 证据: 第1章 科学思维与制度荒诞的碰撞——"一个系统，只要是被设计出来的，就一定有漏洞"（现代工程思维 vs 封建制度的认知摩擦）（章节文本·行为演示）

## 规则 9: 胜利代价律 [演示型规则]

任何正面势力的胜利必须伴随可验证的代价（人员、物质、道德至少两项）。

> ⚠ 演示型规则：证据必须来自章节中具体战斗/事件的结果描写，不得仅引用 world docs。

- 证据: 第2章 田国栋率矿工阻击——"矿工们一个一个倒下。法西斯士兵也在倒下，但倒得更慢。战斗持续了大概一刻钟，矿工这边能站着的只剩五个人"——惨重人员代价（章节文本·行为演示）
- 证据: 第5章 夜袭补给站——"六箱灵能弹匣，一箱医用物资。代价是一个人。"（老李头牺牲）——林烽在账本上的直接计算，人员代价（章节文本·行为演示）
- 证据: 第5章 老李头牺牲后的道德困境——"他不知道自己该不该在账本上写'代价:一个人'。这个人有名字，有孙子，有一个'每次多问一个问题'的习惯"——道德代价的具体呈现（章节文本·行为演示）
- 证据类型: 演示型——所有证据均来自章节中具体战斗的代价描写，无 world doc 引用

## 规则 10: 革命组织发展律 [定义型规则]

林烽领导的革命力量发展必须遵循隐蔽积累 -> 局部斗争 -> 战略反攻的阶段节奏。

- 证据: 第4章 情报网+班组制——"先不打。先看。把矿区里每个工段都找一个能信任的人。村里的也一样。法西斯什么时候换岗、补给车几天来一次、军官住在哪个帐篷、夜里哨兵几个人——这些信息凑够了，我们才能决定怎么打"——隐蔽积累阶段的组织建设（章节文本）
- 证据: 第5章 夜袭补给站——四个人班组、限制行动范围（矿区周边）、以夺取装备而非消灭敌人为目标——局部斗争阶段（章节文本）
- 证据: world-rules.md 规则八——"阶段一（隐蔽积累）：在建立至少30人的核心组织之前，不得进行任何公开行动"（world doc）
- 证据类型: 显性（定义型——阶段框架由 world doc 定义，章节内容展示两个阶段的实践）
