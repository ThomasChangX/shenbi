## 第3卷：燎原之势

**Objective**: 林烽领导的梅德兰抵抗力量是否在第50章结束时完成了从地下隐秘组织到公开人民战争体系的转型——即建立了自给自足的敌后根据地、取得了大规模运动战胜利、并认知升级到"真正的敌人是列塔尼亚殖民框架"？（是/否可判断）

**章节范围**: 第31章 - 第50章（共 20 章）

### Key Results

#### KR1: 建立敌后根据地

- **章节范围**: 第31章 - 第38章（共 8 章）
- **衡量标准**: 事件性
- **对 Objective 的支撑**: 根据地是人民战争体系的物理基础——没有根据地就没有战略纵深
- **章节节点**:
  | 章节 | 节点角色 | 具体定位 |
  |------|---------|---------|
  | 第31章 | 开篇 | 公开第一步：选择南部山区为根据地。先做"最不革命的事"——帮农民算收成、教矿工认字。展示根据地的核心不是军事堡垒，是人心堡垒 |
  | 第32章 | 承接 | 钨灯镇教材化：纪师将50年地下教材转为公开教育体系。林烽贡献现代教学法（分组讨论、实操练习）。展示"知识分子+农民"的结合 |
  | 第33章 | 承接 | 第一场小型保卫战：击退涅普敦散兵试探进攻。运用地形+灵能装置。复盘发现致命问题——装备完全依赖缴获，无生产能力 |
  | 第34章 | 承接 | 滕贵生首次对抗：根据地影响矿脉供应链，工人逃跑加入。滕贵生用经济手段施压——切断粮食药品流通。首次展示经济战 |
  | 第35章 | 承接 | 灵能生产体系I：用分解能力从废料回收制造基本工具。建立根据地的初级回收加工线 |
  | 第36章 | 承接 | 灵能生产体系II：塑形+融合能力制造简易灵能器械。纪师提供梵光时期灵能与农业结合手册 |
  | 第37章 | 承接 | 组织扩展：周边三个村庄主动联系根据地请求保护+加入。展示"根据地不是征服而是吸引"的政治逻辑 |
  | 第38章 | 收官 | 自我造血完成：灵能加速作物在试验田首次成功。根据地实现"可以自己养活自己"的标志性事件 |

#### KR2: 运动战大捷与群众动员

- **章节范围**: 第39章 - 第45章（共 7 章）
- **衡量标准**: 事件性
- **对 Objective 的支撑**: 运动战大捷是"公开人民战争体系"的军事验证——证明根据地不仅能自保，能进攻
- **章节节点**:
  | 章节 | 节点角色 | 具体定位 |
  |------|---------|---------|
  | 第39章 | 开篇 | 麻雀战初试：首次运用十六字诀改编版（灵能战争版）。小规模骚扰战反复成功。展示林烽的战斗哲学——系统工程思维拆解战争 |
  | 第40章 | 承接 | 群众动员深度I：互助组形式编入后勤/情报/建设。铁匠→武器修理，农民→情报网络，纺织工→缝补医疗 |
  | 第41章 | 承接 | 群众动员深度II：矿区工人秘密加入根据地支持网络。展示"人民战争"=让日常技能转化战争资源 |
  | 第42-44章 | 转折 | 第一次运动战大捷：包围歼灭涅普敦一个主力兵团。利用地形（山区）、时间（雨季不熟）、信息优势（群众情报网）。缴获大量列塔尼亚装备含制造编号——这是揭示列塔尼亚军火供应链的关键证据 |
  | 第45章 | 收官 | 胜利后的裂缝：林烽在庆祝会上沉默，老政委追问。胜利表面下不安萌芽——缴获装备数量和编号暗示某种林烽尚未言明的问题 |

#### KR3: 进入更大棋局——觉察列塔尼亚的框架

- **章节范围**: 第46章 - 第50章（共 5 章）
- **衡量标准**: 状态性
- **对 Objective 的支撑**: 觉察列塔尼亚框架是"揭露战争真实规模"的完成——不知道敌人在哪，战争就打错了
- **章节节点**:
  | 章节 | 节点角色 | 具体定位 |
  |------|---------|---------|
  | 第46章 | 开篇 | 数据分析：通过缴获编号反推产量——列塔尼亚每月向涅普敦输送的装备可补充损失速度两倍。列塔尼亚在维持战争，不是支持涅普敦赢 |
  | 第47章 | 承接 | 拉图尔首次关注：列塔尼亚学术会议上，从"梅德兰南部灵能庶民教育率异常上升"数据中注意到林烽。反应不是恐慌——是"这个庶民不符合文明梯度模型"的学术好奇 |
  | 第48章 | 承接 | 格鲁斯塔夫首次重视：审阅战报后愤怒→沉默——林烽的战术不是预期中的"鼠辈打法"。命令参谋："我要林烽的全部情报——不只是军事" |
  | 第49章 | 转折 | 暗流二首次浮现：林烽从夹在货单里的加密文件收到位面通道数学推导（含从未见过的修正因子）。落款符号被纪师认出——梵光时期平等梵天教曾与列塔尼亚某学界支持者秘密通信的暗号。列塔尼亚内部"异端学者"首次暗示 |
  | 第50章 | 收官 | 战争扩大：林烽召集核心成员摊开七位面灵能供应链地图。"如果我们只想打涅普敦——已经赢了。但问题不在涅普敦。问题在那个不出手但确保战争继续的人。"老政委："你知道这意味着什么？"林烽："意味着我们要打的，不是一场战争。"第三卷结束 |

### 与 story_frame 的一致性

- surface_conflict 推进: 从"防御"升级到"进攻"，从"单人"升级到"系统"——根据地建设（抵抗从被动防御转为主动建设生存空间，土地分配和灵能教育本身就是对殖民土地制度和教育垄断的物质反抗）；滕贵生经济战（买办阶级作为殖民三支柱的"买办代理"第一次被物理展示）；运动战大捷（军事上证明梅德兰人民军队可以击败主力兵团，随即被缴获装备编号反噬——胜利同时揭示战争真正的本钱是列塔尼亚军火供应链）
- personal_conflict 推进: 林烽从"革命学徒"逐渐走向"决策者"——第31-34章首次独立做"不革命的事"（帮农民算收成不是老政委教的）；第39章首次运用十六字诀（照搬原版后被纠偏为"因地制宜版"，又一次"第一次笨拙"）；第45章庆祝会上的沉默显示他已完成转折——胜利带来的不是成就感，是更大问题的发现。犬儒主义/躺平/跑路已不再是第一反应
- deep_conflict 推进: 缴获的列塔尼亚装备编号——列塔尼亚"维持战争"的逻辑首次暴露，殖民体系"为谁提供武器、让谁去死"的深层问题；第49章暗流二数学文件——位面通道垄断（技术垄断）的最早裂缝，暗示"打破压迫循环"不是靠道德号召而是靠将技术从垄断中解放；第50章首次提出：革命不是推翻一个政权，而是推翻一套让多个政权在其中运转的体系——深冲突"能否打破压迫循环"的第一次理论接触
