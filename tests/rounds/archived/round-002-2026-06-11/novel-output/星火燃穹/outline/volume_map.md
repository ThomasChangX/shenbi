# 分卷地图

## 第一卷：跪着是活不下去的

**Objective**: 林烽完成从利己主义躺平青年到被动参与革命的转变；梅德兰-涅普敦战争爆发；根据地初建

**Key Results**:
1. KR1：林烽在异世界站住脚（第1-5章完成）
   - 第1章节点（开篇）：穿越、负债、催收——建立异世界底层生存的残酷性
   - 第2章节点（承接）：工厂打工、目睹阶级差距——良知第一次碰撞
   - 第3章节点（转折）：空袭轰炸、工友惨死——利己主义幻灭，灵能觉醒
   - 第4章节点（发展）：逃亡与组织残存平民——第一次展现领导潜力
   - 第5章节点（收官）：遇到贺云山和残存梵天教徒——接触革命的真正历史

2. KR2：根据地初步建立（第6-10章完成）
   - 第6章节点（开篇）：进入焚风荒原——理解梵天教的现状和分裂
   - 第7章节点（承接）：林烽首次运用现代知识解决实际问题（改造灵能冶炼炉效率）
   - 第8章节点（转折）：涅普敦正规军压境——第一次军事危机
   - 第9章节点（发展）：林烽运用毛概思想组织第一次游击战——小胜
   - 第10章节点（收官）：根据地初具雏形——但内部矛盾开始浮现

3. KR3：站稳脚跟（第11-15章完成）
   - 第11章节点（开篇）：投降派试图出卖林烽换取招安
   - 第12章节点（承接）：林烽粉碎内部叛乱——政治成熟的第一步
   - 第13章节点（转折）：列塔尼亚第一次公开干预——以"调停"为名施压
   - 第14章节点（发展）：林烽与阿尔方斯第一次正面交锋——外交战场
   - 第15章节点（收官）：根据地巩固——贺云山正式将密卷交给林烽

**节奏原则**: 第一卷节奏前慢后快——前5章以人物建立和世界观沉浸为主，第6章开始加速进入战争节奏。张力曲线在第3章、第8章、第13章各有一个主要高点。

**跨卷衔接**: 第一卷结尾贺云山交密卷的同时透露：列塔尼亚的真正目标是古通道。第二卷的核心任务从"保卫根据地"升级为"夺取通道控制权"。

## 第二卷：实事求是

**Objective**: 根据地发展为区域级战略力量；林烽确立军事思想；揭露列塔尼亚的双重游戏；与瓦尔特第一次大规模对决

**Key Results**:
1. KR1：根据地扩张（第16-22章完成）
2. KR2：梅德兰军事思想的确立（第23-29章完成）
3. KR3：与瓦尔特的决战（第30-36章完成）
4. KR4：夺取第一个古通道控制权（第37-42章完成）

**节奏原则**: 第二卷全程高强度军事节奏，通过"运动战—反围剿—战略反攻"三阶段递进。每7章一个军事周期。

**跨卷衔接**: 通道被激活后揭示了第一批前文明信息。第三卷的核心任务升级为"解密封印+联盟外交"。

## 第三卷：星火燃穹

**Objective**: 林烽完成向文明哲学家的升华；大决战击败涅普敦主力并逼退列塔尼亚；揭示通道封印真相；提出文明命运共同体构想

**Key Results**:
1. KR1：解读通道信息与封印真相（第43-50章完成）
2. KR2：联合其他位面形成反殖民联盟（第51-58章完成）
3. KR3：史诗级大决战（第59-70章完成）
4. KR4：战后秩序构建与文明命运共同体宣言（第71-78章完成）

**节奏原则**: 第三卷以"揭秘—联盟—决战—建政"四阶段展开。前段信息密度高（解密），中段外交节奏（联盟），后段战争高潮（决战），末段收束（建政）。

**跨卷衔接**: 无下一卷——终场画面收束全书。但留下开放式余韵：新一代梅德兰青年驾驶星舰驶向星辰大海，林烽以普通探索者的身份跟随其中一艘船，消失在位面之门的光芒中。

## 第一卷详细大纲：跪着是活不下去的

**Objective**: 林烽完成从利己主义躺平青年到被动扛起革命责任的转变；梅德兰-涅普敦战争全面爆发；焚风荒原根据地初步建立并经受住首次内外夹击

**章节范围**: 第1章 - 第15章

**二元判断**: 卷末时，林烽是否已主动接受革命责任（而非被动卷入）？根据地是否具备抵御中等规模围剿的能力？——两项均为"是"则本卷完成。

### Key Results

#### KR1：林烽在异世界站住脚，完成从逃避到不得不面对的转变

- **章节范围**: 第1章 - 第5章
- **衡量标准**: 状态性 —— 林烽从"想办法穿回去/独善其身"转变为"这个世界的问题我绕不开"
- **章节节点**:
  - 第1章: [开篇] 穿越附身、负债催收——建立异世界底层生存的残酷规则
  - 第2章: [承接] 工厂打工、目睹种姓隔离——第一次良知碰撞，仍试图用"管好自己"合理化逃避
  - 第3章: [转折] 涅普敦空袭、工友惨死——利己主义幻想被打碎，灵能首次觉醒
  - 第4章: [承接] 带领残存平民逃亡——被动展现领导潜力
  - 第5章: [收官] 遇到贺云山与梵天教徒——首次接触被掩盖的革命历史

#### KR2：根据地初步建立，现代知识与异世界实践的第一次融合

- **章节范围**: 第6章 - 第10章
- **衡量标准**: 事件性 —— 根据地完成首批难民收容 + 实现基本自给自足 + 成功击退第一次扫荡
- **章节节点**:
  - 第6章: [开篇] 进入焚风荒原，了解梵天教现状——投降派渗透，根据地残破
  - 第7章: [承接] 现代知识改造灵能冶炼炉效率——林烽仍视之为"技术扶贫"而非"革命"
  - 第8章: [转折] 涅普敦正规军扫荡——林烽被迫临阵决策
  - 第9章: [承接] 毛概军事思想指导第一次游击战胜利
  - 第10章: [收官] 根据地初具雏形，投降派在内部散布"招安论"

#### KR3：粉碎内外夹击，林烽完成政治成熟第一步

- **章节范围**: 第11章 - 第15章
- **衡量标准**: 关系性与事件性 —— 投降派被清理 + 领导核心形成 + 列塔尼亚干预被顶住
- **章节节点**:
  - 第11章: [开篇] 投降派试图出卖林烽换取招安
  - 第12章: [转折] 粉碎内部叛乱——林烽政治成熟第一步
  - 第13章: [承接] 列塔尼亚公开介入施压
  - 第14章: [转折] 林烽与阿尔方斯第一次外交交锋
  - 第15章: [收官] 贺云山交密卷——林烽正式接棒

### 卷内张力曲线

- **铺垫段 (第1-3章, 20%)**: 张力低起，第1-2章建立日常，第3章空袭形成第一个尖峰
- **上升段 (第4-8章, 33%)**: 张力递增，多重冲突叠加——逃亡→遇贺云山→荒原现状→技术改良→军事危机
- **爆发段 (第9-14章, 40%)**: 卷高潮区——游击战胜利→内部裂缝→投降派叛卖→粉碎叛乱→列塔尼亚干预→外交交锋
- **余波段 (第15章, 7%)**: 张力回落，密卷传承为情感锚点，信息揭示将读者引向第二卷

### 跨卷桥接

- **实体钩子 1 (物品)**: 贺云山交给林烽的密卷——内含梵光关于古通道封印的部分记录
- **实体钩子 2 (信息)**: 列塔尼亚真正目标是激活古通道，熵痕危机是他们的倒计时
- **实体钩子 3 (人物)**: 阿尔方斯确立为第二卷常驻对手

### 与 story_frame 的一致性

- **surface_conflict**: 第3章空袭→第8章扫荡→第13-14章列塔尼亚干预，从两国战争升级为位面格局博弈
- **personal_conflict**: 第1-2章利己苟活→第3-5章良知唤醒→第8-9章指挥责任→第11-12章政治背叛→第15章接棒
- **deep_conflict**: 第2章系统性压迫日常→第5章被篡改的历史→第13-14章帝国主义逻辑→第15章文明存亡命题
