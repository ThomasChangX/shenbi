# 章节摘要追踪

## 第1章：毕业即失业与穿越即负债

**字数**: ~3800字

**核心剧情**: 林烽穿越附身梅德兰帝国地基区底层青年"林"，醒来即面临灵能贷款催收员钱三彪的暴力催收。他用现代人的逻辑思维试图狡黠赖账——列三条论据讲道理——被钱三彪揭露灵能贷款合同的真实条款（利滚利、灵能烙印回收、灵能苦役营）和高等种姓特权豁免权。被壮汉一拳打醒后，林烽第一次清晰认识到异世界底层被系统性剥削的逻辑矛盾：借20金币，灵能值不了1金币，但利滚利永远还不清，而法律上他欠这笔钱。钱三彪留下三个铜币离开后，林烽听到巷子深处传来压抑的哭声，他从泥地上爬起来朝哭声走去——还不知道自己要做什么，但已不再是那个只想躺平的利己主义者。

**登场角色**: 林烽（主线）、钱三彪、两名壮汉打手

**新建立的世界观**:
- 梅德兰帝国、地基区/上城区地理结构
- 灵能体系：种姓决定灵能位置（高等在脑，底层在血）
- 灵能贷款/催收制度：利滚利、灵能烙印、灵能苦役营、回收
- 新富族宪章第48条：高等种姓特权豁免权（列塔尼亚驻军担保）
- 灵能冶炼厂：底层苦力劳动场所

**情绪弧线**: 困惑 → 狡黠 → 暴力打击 → 清醒认知 → 利他转折

**章尾悬念**: 林烽朝巷子深处的哭声走去——读者想知道哭泣者是谁

**新增伏笔**: 高等种姓特权豁免权

## 第2章：霓虹灯下的烂疮

**字数**: 5214字

**核心剧情**: 林烽来到三号灵能冶炼厂打工，遇到老工友孟虎。上午在冶炼线上目睹何监工（高等种姓）因工人受伤而冷漠——工人脸着地磕破鼻梁，监工只看了一眼便下令"记旷工扣三天钱拖废料场"。孟虎解释新富族宪章第48条的实际运作：从立出来至今没有过一件判例，因为只要高等种姓说"不是故意的"，就不是故意的。傍晚放工后林烽尝试用灵能分解废料——通过背诵材料学成分激活知识驱动型感知，成功从废石中分解出铜粉（原理验证）。天黑后去垃圾场翻"死渣"，分解出3粒贵金属颗粒。在一堆矿渣下扒出奄奄一息的少女苏萤——被权贵玩弄后抛弃。林烽理性计算告诉他"埋回去当没看见"是最优解，但站了20秒后无法迈腿——"他的逻辑到此断裂"。他用外衣裹住苏萤带回藏身处，途中捡到一块巴掌大的金属片——三层同心圆+六角星符号铭文，内部结构非天然合金，恒温冰凉（前穿越者梵光政权遗物）。回到藏身处后，林烽盘算：1铜币买杂粮饼+1铜币买止血草，剩1枚铜币。明天钱三彪还会来。

**登场角色**: 林烽（主线）、孟虎（新登场/工友）、何监工（新登场/高等种姓监工）、苏萤（新登场/被救少女）、旁及工人群体、冶炼厂200名苦力

**新建立的世界观**:
- 三号灵能冶炼厂：熵痕工业带，三根烟囱，十二排工作台每排20苦力，手掌贴废矿石感知分解矿物
- 新富族宪章第48条实际运作：高等种姓的"主观恶意"标准从未被界定——事实上享有绝对无罪豁免（无判例）
- 灵能感知的另一条路径：知识驱动型——背诵材料成分可增强感知精度（林烽的材料学知识作为异世界金手指）
- 废料堆灰色地带：废料堆边缘散碎矿渣不算财产（无人计数）
- 垃圾场：工厂死渣堆积地，含纳米级贵金属、稀有元素——工厂粗放筛选筛不出来
- 高等种姓灵能猎犬：值普通工人十年工钱
- 上城区夜生活：管弦乐派对持续两时辰，地基区只能听到"上面在拍手"
- 林烽灵能限制："电池"极易耗尽，每次感知和分解消耗无名能量储备

**情绪弧线**: 务实搞钱 → 目睹阶级差距愤怒 → 短暂工友温情缓冲 → 阶级凝视/虚无 → 灵能探索兴奋 → 发现苏萤震惊 → 理性vs良知冲突（逻辑断裂）→ 良知压倒利己付诸行动 → 生存焦虑收束

**章尾悬念**: 三重复合悬念——金属片铭文身份（信息悬念）+ 苏萤生死（情感悬念）+ 仅剩1铜币明天催收（生存悬念）

**伏笔兑现**:
- 巷子深处哭泣的人 → 部分兑现：苏萤（读者可推断时间地点吻合；林烽未将两者联系）
- hook-001（特权豁免权）→ 轻度advance：通过何监工行为+孟虎解释展示第48条实际运作

**新增伏笔/线索**:
- 金属片铭文（前穿越者梵光政权遗物，放入活跃伏笔池）
- 林烽灵能特征确认：知识驱动型感知分解（金手指核心机制确定）
- 苏萤身份/来历（她是被谁玩弄后抛弃的？为何被丢在垃圾场？）

## 第3章：灰烬的重量

**字数**: 6315字

**核心剧情**: 林烽在重工业镇已生活约半个月——白天跟孟虎去废料场偷残渣、用灵能分解提纯换铜币，苏萤住在孟虎棚屋山墙搭出的斜顶下。清晨三人准备去东区废料场时，孟虎发现天上"黑点"在变大——不是鸟。涅普敦帝国的灵能炸弹降临。第一颗砸在冶炼厂冷却塔，人群涌向镇公署防空洞，但铁栅栏锁死——权贵们正从公署起降平台登上四艘私人灵能飞艇，私兵持灵能长枪挡住五千平民。一个工人砸栅栏被私兵开枪打死，飞艇排成一线向西飞去，铁栅栏钥匙丢在二十步远的台阶上。第三颗炸弹追来，孟虎用两百多斤的身体扑倒林烽和苏萤，仓库砖墙倒塌——砖石碎片从孟虎脊椎与颅骨之间打进去，孟虎在说完"那姑娘——你带她走"后死亡。苏萤左小腿被钢筋穿透，林烽蹲下用灵能分解掰断钢筋，但一截留在腿内。第四颗炸弹擦仓库屋顶落入巷子——苏萤被炸散，只剩嵌在土里的钢筋残段和一片旧褂子布料。林烽的大脑开始列清单——孟虎死了、苏萤死了、你连她是谁都不知道——"清单列到这里，列不下去了"。灵能在愤怒与绝望中失控觉醒：热量从小腹上涌，暗红色光从指尖渗出，砖石熔化从指缝淌下；原身"林"的临终记忆被灵能翻出（弟弟感染死亡，林绝望而死）；第三次灵能释放将半径十几米内瓦砾全部推开。力竭后林烽额头磕进泥土，手指无意识摸到口袋里的金属片——微微发热（被灵能冲击波激活）。一架飞行器低空掠过，一块残片落在十几米外——内侧镶嵌淡蓝色蛇形纹章：两条蛇首尾相接，中间穿过一把钥匙（列塔尼亚纹章）。林烽不认识，但记住了。章末天空被火光烧成铁锈色，上城区反重力阵列纹丝不动，残片上的纹章仍在发光。

**登场角色**: 林烽（主线）、孟虎（牺牲）、苏萤（死亡）、权贵群体/私兵（特权豁免展示）、旁及广场五千平民、涅普敦飞行器编队

**新建立的世界观**:
- 重工业镇地理：工棚区（废料板拼建）、东区废料场、镇公署广场/防空洞（容纳3000人，镇上住20000人）、仓库窄巷、灵能飞艇起降平台
- 涅普敦帝国空袭：灵能炸弹从天垂直劈下（先白光再转橘红），冲击波造成气浪和砖石倒塌；飞行器编队从东边压来
- 高等种姓特权豁免的暴力延伸：空袭中权贵乘私人灵能飞艇优先逃生，平民被锁防空洞栅栏外；私兵对砸栅栏者开枪；钥匙丢在够不到的地方
- 灵能觉醒机制：可在极端情绪（愤怒+绝望）下失控觉醒；非升级/突破——是心智跟不上能力的失控状态（"三岁小孩被塞进踩满油门的卡车"）；觉醒后身体被掏空
- 原身记忆机制：灵能冲击波可翻出被压在大脑最底层的记忆碎片（"林"的临终记忆）
- 列塔尼亚纹章首次展示：两条蛇首尾相接衔一把钥匙，蛇眼紫色宝石，钥匙柄藤蔓状未知文字；淡蓝色荧光镶嵌在金属中；出现在涅普敦飞行器残片内侧

**情绪弧线**: 平静日常（算账思维）→ 空袭惊吓 → 回头救人的本能 → 孟虎死亡的冲击 → 苏萤死亡的绝望 → 利己主义彻底崩塌 → 灵能在愤怒中失控觉醒 → 力竭后的空洞

**章尾悬念**: 四重复合悬念——列塔尼亚蛇形纹章身份（信息悬念，连接空袭与幕后操纵）+ 金属片被激活发热（伏笔悬念）+ 林烽力竭暴露在持续轰炸中（生存悬念）+ "他不认识它。但总有一天他会认识的。"（命运悬念）

**伏笔兑现**:
- hook-001（高等种姓特权豁免权）→ advance: 从钱三彪口述→亲眼目睹权贵飞艇逃离、平民被锁、私兵开枪（第1章 subtlety 0.5 → 第3章 subtlety 0.8）
- 战争降临 → 兑现: 从第1章开始的铺垫（列塔尼亚驻军担保、帝国紧张局势的暗示）→ 第3章涅普敦空袭真实降临
- 灵能觉醒 → 兑现: 从第1章的"灵能未觉醒（无位者）"→ 第3章在愤怒中失控觉醒（初觉者等级1）
- 金属片 → advance: 第2章在垃圾场发现（恒温冰凉）→ 第3章被灵能冲击波激活（口袋中微微发热，林烽未察觉）
- 苏萤弧光 → 完成: 沉默→"换什么"→"走"→死亡（从被救者到承担者）

**新增伏笔/线索**:
- 列塔尼亚蛇形纹章（飞行器残片——连接涅普敦空袭与列塔尼亚幕后操纵，B1线起点）

- 金属片激活（被灵能冲击波触发发热——梵光遗物功能解锁的第一步）
