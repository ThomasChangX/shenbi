# 角色关系矩阵

## 已登场角色

### 林烽（"林"）

- 身份: 地球中国计算机系毕业生，穿越附身到梅德兰帝国地基区底层青年"林"
- 年龄: 地球21岁 / 梅德兰身体19岁
- 种姓: 底层（灵能在血里）
- 职业: 灵能冶炼厂苦力（第1章中），日薪6铜币
- 灵能等级: 未觉醒（无位者）
- 核心特征: 现代人的逻辑思维习惯，利己主义底色，但在章末开始转变

### 钱三彪

- 身份: 灵能贷款催收员（新富族催收体系中的职业催收人）
- 年龄: 约四十来岁
- 外貌: 胖子，生意人标准的三分笑意，眼神不笑
- 特征: 催收专业，语调和气但内容狠辣；樟脑和皮革气味
- 所属: 两名壮汉的上司
- 装备: 灵能刻板（用于查阅合同/记录）

### 钱三彪手下（2名壮汉）

- 身份: 催收打手
- 外貌: 统一黑色短褂，袖口镶红线
- 特征: 专业暴力执行者——无预兆出拳，接住倒下姿势按墙

### 孟虎

- 身份: 三号灵能冶炼厂老工人，林烽（"林"）的工友
- 年龄: 约四十出头
- 外貌: 壮汉，脸上被炉火烫出的疤从左颧骨拉到下颌，笑起来像脸上多了一道褶子
- 种姓: 底层
- 特征: 粗犷直接，口头禅"你小子"；在冶炼厂干了五年，是能把灵能感知变成呼吸一样本能的老手；知道厂里各种灰色地带（哪个废料堆没人管、哪儿的水不能喝）；把水烧过后分给林烽；看似粗糙但有关切——拍肩提醒监工心情不好
- 位置: 第七排第三个工作台隔壁（第七排林烽工位旁）

### 苏萤

- 身份: 被权贵玩弄后抛弃的少女（具体身份不明）
- 年龄: 约十四五到十六岁（营养不良可能使判断不准）
- 外貌: 身穿曾经可能是淡蓝色的连衣裙（已被撕扯不成形，左袖从肩膀处裂开）；手臂和肩膀上分布深浅不一淤青（新旧交叠）；左眼眶肿得只剩一条缝；嘴唇干裂；严重营养不良（轻得不正常）
- 种姓: 不明（身穿丝绸/细棉面料，原属上城区；现被抛弃于垃圾场）
- 特征: 创伤状态——语言极少且破碎（本章无台词）；外伤严重（多处淤青、化脓伤口）；濒死（呼吸极浅极慢，"像一根蜡烛烧到底时的那点光"）
- 位置: 林烽藏身处土坯墙墙角（被林烽从垃圾场带回）

### 何监工

- 身份: 三号灵能冶炼厂监工
- 年龄: 中年
- 外貌: 穿深蓝长袍（丝绸面料，在暗橙炉光里泛柔和反光——与车间200件粗麻布补丁形成对比）
- 种姓: 高等种姓（享新富族宪章第48条特权豁免）
- 特征: 官僚化冷语言；对工伤工人冷漠（"记旷工。扣三天工钱。拖到废料场"）；养有灵能猎犬（价值普通工人十年工钱），跑丢后心情不好
- 位置: 三号灵能冶炼厂巡视廊（高处俯视车间）

- 灵能等级: ~~未觉醒（无位者）~~ → **初觉者等级1**（第3章在愤怒中失控觉醒）

#### 孟虎 — 状态更新: **已死亡**

- 死亡时间: 第3章（空袭中）
- 死亡原因: 用身体为林烽和苏萤挡住仓库砖墙倒塌的砖石——砖石碎片从脊椎与颅骨之间斜着打进后颈；肋骨多根断裂
- 遗言: "那姑娘——你带她走。"
- 死亡描写方式: 眼睛里的光像蜡烛烧到底，火苗颤了一下，灭了
- 角色弧光总结: 粗犷工友 → 灰色地带生存智慧（五年工厂经验）→ 用身体护住两个比自己弱小的人（利他牺牲的最终形态）

#### 苏萤 — 状态更新: **已死亡**

- 死亡时间: 第3章（空袭中）
- 死前状态: 左小腿被钢筋穿透（从小腿正面扎入后面穿出），林烽用灵能分解折断钢筋但一截留在腿内
- 死亡原因: 第四颗灵能炸弹擦仓库屋顶落入巷子——直接炸中苏萤所在位置，遗体被炸散（仅余嵌在土里的钢筋残段和一片旧褂子布料）
- 遗言: "走。"（一个字——从被救者到主动承担者的弧光完成）
- 角色弧光总结: 沉默观察（创伤状态）→ 问"你救我换什么"（评估者）→ 说"走"（承担者）→ 被炸散（死亡有自己的重量，非"激励工具人"）
