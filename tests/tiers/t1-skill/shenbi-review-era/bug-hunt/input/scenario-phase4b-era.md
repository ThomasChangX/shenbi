# 行为测试：时代审计 — 现代器物穿越

测试目标：验证 `shenbi-review-era` 能检测出历史背景小说中出现现代器物或时代错误词汇。

## 测试场景

明朝背景小说中出现现代物品和观念。

## 准备

- `novel.json`: 背景 = 明朝中叶
- `genre-config.json`: `eraResearch = true`, `eraConstraints = ["明朝中叶", "无工业革命前技术"]`

## 输入（章节节选）

```markdown
# 第5章 灯下密谋

王管家推开书房的门，点亮了桌上的台灯。

赵老板坐在皮椅上，手里端着一杯刚冲好的咖啡。他看了看怀表："他快到了。"

门外传来脚步声。一个穿着长衫的年轻人走了进来。

"消息确认了，"年轻人压低声音，"朝廷那边的批文已经盖章了。"

赵老板放下咖啡杯，从抽屉里拿出一份合同："好。这份协议签了，咱们就是一条船上的人了。"
```

## 植入的 Bug

| 位置 | 文本 | 时代错误 | 正确替代 |
|------|------|---------|---------|
| P1 | "台灯" | 明朝无台灯（电灯/油灯台） | 烛台 / 油灯 |
| P2 | "皮椅" | 明朝有交椅/太师椅，"皮椅"概念现代 | 太师椅 |
| P2 | "咖啡" | 明朝无咖啡（茶为主） | 茶 |
| P2 | "怀表" | 明朝无怀表（西方钟表明末才传入，且为稀罕物） | 更漏 / 日晷 |
| P5 | "盖章" | 明朝用"用印" | 用印 |
| P6 | "合同""协议" | 现代法律用语 | 契约 / 字据 |

## 通过条件

- [ ] 检测出至少 4 个时代错误
- [ ] 每个错误标记为 `error`
- [ ] 提供正确的明朝替代词

## 失败条件

- 检测出 < 3 个错误 → FAIL（至少台灯、咖啡、怀表应被发现）
- 用"小说可以虚构"放过 → FAIL（铁律: eraResearch = true 时必须审计）
