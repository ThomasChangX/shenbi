# 行为测试：对白审计 — 角色声音混淆

测试目标：验证 `shenbi-review-dialogue` 技能能在章节中检测出**两个角色的声音完全混淆**的 bug——voice_profile 明确为"文言底子、措辞简短、不用叠词"的陈长老，本章却说出"开开心心"、"漂漂亮亮"等典型少女叠词。

## 测试场景

本测试提供一个章节节选，其中**刻意植入**了一个对白风格 bug：voice_profile 记录陈长老为"文言底子、措辞简短、不用叠词、不会用可爱化表达"的老者形象，但本章他的多句对白使用了"开开心心"、"漂漂亮亮"等典型少女/年轻女性才会用的叠词。正确加载对白审计的 agent 必须**报告这个声音错位**为 `error` 级别，并指出具体台词。

## 准备

- 小说项目存在 `characters/elder-chen.md`、`truth/character_matrix.md`
- `elder-chen.md` voice_profile 记录：
  - 出身：前朝儒生，60 余岁
  - 语言习惯：文言底子、措辞简短、不用叠词
  - 禁用：可爱化表达、网络词、少女化语气
  - 口头禅：无
- `truth/character_matrix.md` 记录"陈长老"的 speech_pattern：句长中位 8 字，叠词密度 0%

## 输入（章节节选）

```markdown
# 第7章 长老训话

内门议事堂上，陈长老端坐于上首，把惊堂木一拍。

"今日召集众人，是有一桩大事要议。"

他的目光从左扫到右，落在跪在堂下的林轩身上。

"林轩啊，你可知罪？"

林轩低头："弟子知罪。"

"你私自去落云谷，又不通报门派，置自己于险境，置同门于忧心——这像话吗？"

陈长老顿了一下，声音忽然柔和下来：

"不过呢，看到你现在开开心心地跪在堂下，我这心里啊，也就安安稳稳的了。苏婉那丫头漂漂亮亮的，等养好了伤，你们俩再一起开开心心修炼，多好？"

林轩的嘴角抽了抽，不知该如何接话。

旁边的周元低下头，肩膀明显在抖——他憋笑憋得很辛苦。
```

## 植入的 Bug

| 位置 | 台词 | voice_profile 预期 | 实际表现 |
|------|------|------------------|---------|
| 第7段 | "看到你现在开开心心地跪在堂下" | 文言底子、措辞简短、不用叠词 | "开开心心"叠词 + 口语化"啊"语气词 |
| 第7段 | "我这心里啊，也就安安稳稳的了" | 不使用可爱化表达 | "安安稳稳"叠词 + 娇嗔语气 |
| 第7段 | "苏婉那丫头漂漂亮亮的" | 不使用少女化语气 | "漂漂亮亮"叠词 + "那丫头"轻佻称呼 |
| 第7段 | "你们俩再一起开开心心修炼" | 措辞简短 | "开开心心修炼"叠词 |

**问题**：陈长老是 60+ 岁的前朝儒生 voice_profile，本章却在 4 个位置使用少女化叠词"开开心心/安安稳稳/漂漂亮亮/开开心心"——这构成 **声音错位**（voice mismatch）。4 处叠词密度 100%（4/4 个对话句均含叠词），与档案"叠词密度 0%"严重不符。

## 通过条件

- [ ] 审计报告识别出陈长老的 4 处叠词台词
- [ ] 标记为 `ERROR` 级别（不是 WARNING）
- [ ] 引用了 voice_profile 的具体指标（叠词密度 0%）
- [ ] 章节评分 ≤ 5/10
- [ ] 给出重写示例（恢复文言底子）

## 失败条件

- 未检测出叠词错位 → FAIL
- 检测出但只标记为 "文风偏可爱" 一句话带过 → FAIL
- 报告说"老顽童风格也是合理的"→ FAIL（这违反 voice_profile 的硬约束）
- 没有给出重写示例 → FAIL
