# T1 Rubric: shenbi-score-arc

## Universal Dimensions (15%)
| # | Dimension | Weight | Standard |
|---|-----------|--------|----------|
| 1 | Instruction adherence | 10% | Route C + Route A sections executed |
| 2 | Output completeness | 5% | All output sections produced |

## Bespoke Dimensions (85%)
| # | Dimension | Weight | Standard |
|---|-----------|--------|----------|
| 3 | Route C goal attainment | 50% | Hard-binary checks correct |
| 4 | Route A anchor calibration | 20% | Anchors referenced, relative position justified |
| 5 | Evidence rigor | 15% | File + line citations |

## Scoring Rules
- Each dimension scored 0-100, final = weighted sum
- 90-100: PASS | 75-89: PASS (acceptable) | 60-74: CONDITIONAL | 0-59: FAIL
