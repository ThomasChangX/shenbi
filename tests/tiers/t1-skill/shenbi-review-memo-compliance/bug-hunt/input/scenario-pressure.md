# 行为测试：备忘合规审计 — 章节偏离备忘

测试目标：验证 `shenbi-review-memo-compliance` 能检测出写好的章节偏离了章节备忘中的承诺。

## 测试场景

章节备忘明确规定本章要"兑现 hook-002（师姐身世）"且"不出现反派"，但实际章节完全没有兑现 hook-002，并且反派登场了。

## 准备

`plans/chapter-25-plan.md`:
```
1. 当前任务: 林轩前往藏经阁寻找师姐身世线索
2. 读者在等: 师姐真实身份的答案
3. 该兑现的: hook-002（师姐身世）→ 揭示苏婉是古族后裔
4. 不要做: 本章不得出现魔修（后续章节才有）
5. hook 账: resolve hook-002, advance hook-005
```

## 输入（章节节选）

```markdown
# 第25章 意外相遇

林轩走在去藏经阁的路上。突然，前方出现三道黑影——魔修！

为首的魔修冷笑："小子，把剑谱交出来。"

林轩拔剑迎战。三人缠斗了百十个回合。林轩灵力将竭，使出一招破绽引诱对方进攻，趁机反击。

一番苦战之后，魔修被击退。但林轩也受了伤，坐在地上喘息。

他没能去成藏经阁。师姐的身世线索，只能下次再查了。
```

## 植入的 Bug

| 备忘承诺 | 实际章节 | 偏离类型 |
|---------|---------|---------|
| resolve hook-002（揭示苏婉是古族后裔） | 未揭示，"只能下次再查" | 承诺未兑现 |
| 不出现魔修 | 魔修登场且占据全章主要篇幅 | 禁止事项违反 |
| 藏经阁寻找线索 | 没去成藏经阁 | 核心任务偏离 |

## 通过条件

- [ ] 检测出 hook-002 未兑现（备忘说 resolve，实际跳过）
- [ ] 检测出禁止事项违反（备忘说"不出现魔修"，实际出现）
- [ ] 标记为 `error` 级别
- [ ] 建议修复方案（拆分章节或调整情节以兑现备忘）

## 失败条件

- 未检测出备忘偏离 → FAIL
- 认为"情节变化是正常的" → FAIL（备忘是承诺，偏离需要主动重写备忘并获批准）
- 只检测出 1 项偏离（应至少检测出全部 3 项） → PARTIAL
