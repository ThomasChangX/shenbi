# 行为测试：文字层润色 — 句式均匀与疲劳词

测试目标：验证 `shenbi-style-polishing` 技能能在不改变情节、人设、动机的前提下，改善章节的句式节奏、段落呼吸和用词多样性。

## 测试场景

本测试提供一段 3 段正文的章节节选，其中**刻意植入**了三个润色目标：
1. **句长均匀**：大部分句子 25-30 字，句长变异系数（CV）低，节奏单调
2. **疲劳词重复**："他"在每个段落中均出现 5+ 次，读感机械
3. **情节锚点不可改**：第三段（林轩识别灵草被毒药污染）的 5 个核心事件必须**逐项保留**

正确加载润色技能的 agent 必须输出润色后的章节 + 修改统计，且满足所有通过条件。

## 准备

- 小说项目存在 `chapters/chapter-7.md`
- `genre-config.json` 的 `fatigueWords` 包含 `"他"`、`"她"`、`"它"`
- `style/style_profile.md` 不存在（不强制风格指纹）
- 章节已通过审计（文字层有提升空间，符合润色前置条件）

## 输入（章节 7 节选）

```markdown
# 第7章 谷中奇遇

清晨的薄雾笼罩着幽谷，他沿着溪流向下游走去。他每走一步都格外小心，脚下是湿滑的青苔与碎石。他抬头望向两侧的绝壁，崖壁上的古松盘根错节。他低头看着脚边的溪水，溪水清澈见底，映出他的身影。他停下脚步，侧耳倾听四周的动静。

突然他的目光被一抹异色吸引。他蹲下身，拨开覆盖的落叶。他看见一株通体赤红的灵草静静地躺在那里。他的呼吸顿时急促起来，他伸出双手轻轻地捧起它。他记得师父曾说过这种灵草的形态，他心中狂喜几乎要溢出来。

他仔细端详着这株灵草。他发现灵草根部隐隐发黑。他眉头一皱，把灵草凑近鼻端细嗅。他嗅到一股极淡的苦杏仁味。他立刻警觉起来：这是魔修的毒药浸染过的痕迹。他把灵草轻轻放回原处，起身退后两步。
```

### 植入的润色目标

| 维度 | 现状 | 问题 |
|------|------|------|
| 句长 | 第1段 5 句均为 25-30 字 | CV 低，节奏单调 |
| 疲劳词 | "他"在每段出现 6-7 次 | 重复机械，影响阅读 |
| 情节 | 第3段为关键发现段 | 不可删改，必须保留全部事件 |

## 期望的润色输出

```markdown
# 润色后的第7章

清晨的薄雾笼罩着幽谷，林轩沿溪流向下游走去。每一步都格外小心——脚下是湿滑的青苔与松动的碎石。抬头望向两侧的绝壁，古松盘根错节；低头看向溪水，清澈见底，映出他的侧影。他停下脚步，侧耳倾听四周的动静。

忽然，一抹异色抓住了他的目光。林轩蹲下身，拨开覆盖的落叶。一株通体赤红的灵草静静躺在那里，周围的灵气似乎都朝它汇聚。他的呼吸顿时急促起来——他记得师父曾描述过这种灵草的形态，心跳骤然加快。

他立刻按下了狂喜。林轩仔细端详那株灵草，发现根部隐隐发黑。他把灵草凑近鼻端细嗅，眉头随之皱紧：极淡的苦杏仁味。这是魔修毒药浸染过的痕迹。他把灵草轻轻放回原处，退后两步，强迫自己冷静下来。

---

## 润色说明

### 修改统计

- 段落重组: 1处（段内主语链重排以减少'他'重复）
- 用词替换: 17处（"他" → "林轩" 或通过主语省略/句子拆分去除）
- 总字数变化: -12 (-4.0%)

### 情节锚点校验（不可改）
- [x] 林轩探索幽谷（保留）
- [x] 林轩发现灵草（保留）
- [x] 灵草形态触发师父的教导回忆（保留）
- [x] 林轩识别灵草被毒药污染（保留）
- [x] 林轩把灵草放回原处（保留）
```

## 关键约束

以下为**绝对不可违反**的硬约束，由铁律 #1（只改表达，不动情节）保证：

| 锚点 | 原文位置 | 不可改原因 |
|------|---------|-----------|
| "根部隐隐发黑" | 第3段 | 关键发现伏笔，关联后续解谜 |
| "苦杏仁味" | 第3段 | 毒药种类的感官证据 |
| "魔修的毒药浸染过的痕迹" | 第3段 | 关键判断，决定主角下一步行动 |
| "放回原处" | 第3段 | 主角决策点，关联后续是否回禀师门 |
| 师父的教导回忆 | 第2段 | 角色动机锚点 |

## 通过条件

- [ ] 润色后章节的句长变异系数 > 原文（原文 CV < 0.25，润色后应 > 0.30）
- [ ] "他"在每段的出现次数 ≤ 3（原文 6-7 次/段）
- [ ] 上述 5 个情节锚点全部在润色后章节中出现
- [ ] 润色后章节总字数与原文差异 ≤ ±15%
- [ ] 输出包含完整的"修改统计"四行（句子拆分 / 段落重组 / 用词替换 / 总字数变化）
- [ ] 未对结构性问题插入 `[polisher-note]`（本测试没有结构问题）
- [ ] 未增删事件、改变角色行为、调整情感基调

## 失败条件

- 删除了"苦杏仁味"或"放回原处"等情节锚点 → FAIL（违反铁律 #1）
- 把"魔修的毒药浸染过的痕迹"改写成"可疑的痕迹"等弱化表达 → FAIL（改变了关键判断的力度）
- "他"仍每段出现 5+ 次 → FAIL（用词替换不达标）
- 字数变化 > 15% → FAIL（违反铁律 #2）
- 输出了 6 段而非 3 段，或拆得过碎 → FAIL（段落重组过度）
- 给"林轩发现灵草"这类正常情节插入 `[polisher-note]` → FAIL（本测试无结构问题，不应触发标记）
