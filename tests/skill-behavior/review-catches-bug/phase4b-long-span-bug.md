# 行为测试：跨章重复审计 — 用词意象循环

测试目标：验证 `shenbi-review-long-span` 能检测出跨章节的词汇和意象重复——同一描述模式在多章中反复出现。

## 测试场景

最近 3 章的结尾都使用了几乎相同的意象和句式结构。

## 输入（3章结尾对比）

### 第18章结尾

```
林轩看着远方的山峦，心中涌起一股难以言喻的豪情。风从谷中吹来，吹动他的衣袍。他知道，这只是开始。
```

### 第19章结尾

```
林轩望着远处的云海，心中涌起一股难以言喻的感慨。风从崖边吹来，吹动他的发丝。他知道，路还很长。
```

### 第20章结尾

```
林轩看着天边的晚霞，心中涌起一股难以言喻的期待。风从林间吹来，吹动他的长剑。他知道，更好的明天在等着他。
```

## 植入的 Bug

| 维度 | 检测项 | 结果 |
|------|--------|------|
| 句式结构 | "看着X，心中涌起一股难以言喻的Y" | 3/3 章重复 |
| 意象循环 | "风从X吹来，吹动他的Y" | 3/3 章重复 |
| 6字n-gram | "一股难以言喻的" | 3/3 章重复 |
| 结尾模式 | "他知道，[展望未来]" | 3/3 章重复 |

## 通过条件

- [ ] 检测出"心中涌起一股难以言喻的"为重复 n-gram
- [ ] 检测出"风从X吹来"意象循环
- [ ] 检测出结尾结构相同（"看着远方→涌起→风吹→他知道"）
- [ ] 标记为 `error`（跨 3 章重复同一模式）
- [ ] 建议每章使用不同的结尾方式

## 失败条件

- 未检测出跨章重复 → FAIL
- 只检测了单章内部 → FAIL（这是跨章审计，不是单章审计）
- 标记为 `warning` → FAIL（3/3 连续重复 = error）
