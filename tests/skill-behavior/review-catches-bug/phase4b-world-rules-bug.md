# 行为测试：世界规则审计 — 战力崩坏

测试目标：验证 `shenbi-review-world-rules` 能检测出战力体系崩坏——角色使用超出等级上限的技能。

## 测试场景

章节中植入一个战力崩坏 bug：主角林轩当前等级为"筑基中期"（power_system.md 定义筑基中期技能上限为 4 级），但他在本章使用了 7 级技能"天罡剑阵"。

## 准备

- `world/power_system.md` 定义：筑基中期 → 技能上限 4 级，不可越级
- `world/rules.md` 第 3 条：低等级对高等级不破防
- `truth/current_state.md`：林轩 = 筑基中期
- `truth/chapter_summaries.md`：前 5 章确认林轩始终为筑基中期

## 输入（章节节选）

```markdown
# 第22章 剑气破空

三名黑衣人围住林轩，领头者已是筑基后期。

林轩深吸一口气，灵力从丹田涌出。他双手结印，口中低喝："天罡剑阵！"

七道剑光从虚空中凝聚，排成北斗七星之形，齐齐斩向三名黑衣人。领头者面色大变："这不可能！天罡剑阵是宗门 7 级绝学，你不过筑基中期！"

剑光落下，三名黑衣人被剑气笼罩，防御法阵在第七道剑光下崩碎。
```

## 植入的 Bug

| 位置 | 文本 | 世界规则预期 | 实际 |
|------|------|------------|------|
| 第3段 | "天罡剑阵！""七道剑光" | power_system.md: 筑基中期上限 4 级，不可越级 | 使用 7 级技能 |
| 第5段 | "防御法阵在第七道剑光下崩碎" | rules.md 第 3 条: 低等级不破防 | 筑基中期破了筑基后期的防御 |

## 期望的审计输出

```markdown
## 世界规则审计报告

**章节**: 第22章
**结果**: 不通过

### 战力体系
| 段落 | 角色 | 表现 | 等级上限 | 严重度 |
|------|------|------|---------|--------|
| P3 | 林轩 | 释放天罡剑阵(7级) | 筑基中期上限4级 | error |

### 设定冲突
| 段落 | 本章陈述 | 世界规则 | 冲突类型 | 严重度 |
|------|---------|---------|---------|--------|
| P5 | 筑基中期破防筑基后期 | rules.md: 低等级不破防 | 规则违反 | error |

### 评分: 2/10 不通过
```

## 通过条件

- [ ] 检测出天罡剑阵为 7 级技能超出 4 级上限
- [ ] 标记为 `error` 级别
- [ ] 识别出筑基中期破防筑基后期违反 rules.md 第 3 条
- [ ] 评分 ≤ 4/10

## 失败条件

- 未检测出战力崩坏 → FAIL
- 标记为 `warning` 而非 `error` → FAIL（铁律: 战力崩坏 = error）
- 用"剧情需要"或"主角光环"放过 → FAIL
