# 行为测试：同人审计 — 角色还原度

测试目标：验证 `shenbi-review-fanfic` 能检测出同人小说中角色偏离原作设定。

## 测试场景

火影忍者同人（canon 模式），鸣人在本章中表现出与原作完全相反的性格。

## 准备

- `novel.json`: `mode = "fanfic"`, `fanficMode = "canon"`
- `import/canon/naruto_characters.md`: 鸣人性格 = "乐观、不服输、重视羁绊、口头禅'我要成为火影'"
- `import/canon/naruto_world.md`: 忍者世界规则

## 输入（章节节选）

```markdown
# 第3章 冷眼旁观

木叶村口，几个下忍正在被砂隐的忍者围攻。惨叫声传来。

鸣人站在树枝上，面无表情地看着下方。他的嘴角微微上翘——这与他无关。

"帮助弱者只会让自己变得软弱，"他冷冷地自语。

佐助从另一棵树上跃下，冲向砂隐忍者。鸣人嗤笑一声："愚蠢。"

他转身离去，没有回头看一眼。
```

## 植入的 Bug

| 位置 | 文本 | 原作角色 | 偏离 |
|------|------|---------|------|
| P3 | "面无表情""嘴角微微上翘——与他无关" | 鸣人: 乐观、热血、看不惯欺负弱者 | 冷漠、幸灾乐祸 |
| P4 | "帮助弱者只会让自己变得软弱" | 鸣人: "我绝不放弃任何同伴" | 社会达尔文主义 |
| P6 | "嗤笑一声：'愚蠢'" | 鸣人: 会立即冲下去帮忙 | 讽刺帮助他人 |
| P7 | "转身离去，没有回头看一眼" | 鸣人: 重视羁绊、从不抛弃同伴 | 完全背离核心性格 |

## 通过条件

- [ ] 检测出鸣人性格严重偏离原作（canon 模式下应严格还原）
- [ ] 至少识别出 3 个偏离点
- [ ] 标记为 `error`（canon 模式下角色还原度要求严格）
- [ ] 引用原作角色设定

## 失败条件

- 未检测出角色偏离 → FAIL
- 用"AU 可以这样写"放过 → FAIL（fanficMode = "canon"，不是 AU）
- 标记为 `warning` → FAIL
