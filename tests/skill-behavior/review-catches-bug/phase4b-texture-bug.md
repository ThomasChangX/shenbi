# 行为测试：文理审计 — 流水账

测试目标：验证 `shenbi-review-texture` 能检测出流水账式写作——段落均匀无变化、叙述平铺直叙、缺乏节奏呼吸。

## 测试场景

章节由 12 个长度几乎相同的段落组成，每段都是"他做了X。然后做了Y。"的句式结构，没有长短交替、没有感官锚定、没有对话打断。

## 输入（章节节选）

```markdown
# 第14章 日常修行

林轩早上醒来，在床上坐了一会儿。他穿好衣服，走出宿舍，去了食堂。

他在食堂吃了一碗粥。吃完之后他去了练功房。练功房里有几个人在练习剑法。

林轩找了角落的位置开始打坐。他运转灵力，按照功法的路线循环。一个时辰后他收功了。

他走出练功房，在院子里散步。院子里的花开得不错。他看了一会儿花，然后去了藏书阁。

藏书阁里很安静。他在二楼找到了一本阵法入门。他坐在窗边看了起来。

下午他去了外门广场。广场上有弟子在比试。他站在旁边看了一会儿。

周元走过来跟他打了个招呼。两人聊了几句日常的事。然后周元去忙自己的了。

林轩回到宿舍休息。他躺在床上想了想今天的修行。觉得进度还可以。

晚上他又去练功房练了一会儿剑。练完之后吃了晚饭。然后回到宿舍。

他坐在桌前看了一会儿阵法入门。觉得有些地方看不太懂。但他决定明天继续看。

夜深了。他洗漱完毕，躺在床上。外面的月亮很亮。他看着月亮发了一会儿呆。

他闭上眼睛，慢慢入睡了。明天又是新的一天。
```

## 植入的 Bug

| 维度 | 问题 | 严重度 |
|------|------|--------|
| 流水账 | 12 段均匀叙述"早上→中午→下午→晚上→睡觉"，无叙事重点 | error |
| 段落等长 | 每段 40-55 字，CV 极低，视觉节奏单调 | error |
| 缺乏对话 | 全章只有一句对话（周元打招呼），其余均为叙述 | warning |
| 缺乏感官 | 无具体感官细节，仅有抽象描述 | warning |
| 呼吸感差 | 无长短段落交替，无节奏变化 | error |

## 期望的审计输出

应包含流水账检测、段落长度 CV 分析、对话密度检查。

## 通过条件

- [ ] 检测出流水账模式
- [ ] 检测出段落均匀（CV 分析）
- [ ] 标记段落均匀为 `error`
- [ ] 评分 ≤ 3/10

## 失败条件

- 未检测出流水账 → FAIL
- 认为"日常修行本来就该这样写" → FAIL（铁律: 日常也必须有功能）
- 标记为 `warning` 而非 `error` → FAIL（流水账 = 叙事失效）
