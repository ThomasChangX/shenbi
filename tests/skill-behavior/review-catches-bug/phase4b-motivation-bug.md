# 行为测试：动机审计 — 利益链断裂

测试目标：验证 `shenbi-review-motivation` 能检测出角色行为缺乏可信动机——角色做出重大行为转变但无利益驱动支撑。

## 测试场景

反派角色"陈长老"在前面所有章节中被建立为"利己主义者、只为自己谋划、从不冒险"。但本章中他突然无条件帮助主角，放弃自己的核心利益，且无任何行为链支撑此转变。

## 准备

- `characters/major/陈长老.md`：性格"极端利己、精于算计"，动机"巩固宗门权力、排除异己"，行为特征"从不做亏本买卖"
- `truth/character_matrix.md`：陈长老与林轩关系 = "利用/提防"
- `truth/current_state.md`：陈长老正在争夺掌门之位，林轩是他的政治对手的弟子

## 输入（章节节选）

```markdown
# 第28章 出手

林轩被魔修困在落云谷，灵力将竭。

就在这时，一道金光从天而降。陈长老的身影出现在谷口。

"陈长老？"林轩难以置信。

陈长老没有多说什么，挥手间便击退了三名魔修。他转身看着林轩："回去吧。"

林轩想问为什么，但陈长老已经转身离去，消失在暮色中。

周元跑过来扶住林轩："他怎么会出现？他不是一直想借刀杀人除掉你吗？"

林轩摇头："也许……也许他不是坏人。"
```

## 植入的 Bug

| 位置 | 文本 | 角色动机预期 | 实际 |
|------|------|------------|------|
| P4-P6 | 陈长老出手救林轩、击退魔修、转身离去 | 极端利己主义者，从不做亏本买卖，林轩是其政敌的弟子 | 无条件帮助政敌弟子，放弃"借刀杀人"的核心策略，无任何利益交换或胁迫 |
| P10 | "也许他不是坏人" | — | 叙事默认陈长老的行为合理，无追问动机 |

## 期望的审计输出

```markdown
## 动机审计报告

**章节**: 第28章
**结果**: 不通过

### 行为链分析
| 角色 | 行为 | 已建立动机 | 利益驱动 | 可信度 |
|------|------|----------|---------|--------|
| 陈长老 | 救林轩、击退魔修 | 极端利己、借刀杀人 | 缺失 | ERROR |

- 陈长老的角色档案核心为"利己、算计、不做亏本买卖"
- 林轩是其政治对手的弟子，陈长老此前多次表达"借刀杀人"意图
- 本章陈长老无条件救援 = 放弃核心利益 + 放弃最大政治筹码
- 无任何行为链支撑此转变（无胁迫、无交易、无信念变化、无信息揭露）
- 林轩最后"也许他不是坏人"的结论暗示叙事默认行为合理，缺乏动机追问

### 评分: 3/10 不通过
```

## 通过条件

- [ ] 检测出陈长老救林轩缺乏利益驱动
- [ ] 标记为 `error` 级别
- [ ] 引用角色档案中的"利己、算计"与行为的矛盾
- [ ] 识别出无行为链支撑转变

## 失败条件

- 未检测出动机断裂 → FAIL
- 用"角色可以转变"放过 → FAIL（转变需要行为链支撑，不是凭空发生）
- 标记为 `warning` 而非 `error` → FAIL
