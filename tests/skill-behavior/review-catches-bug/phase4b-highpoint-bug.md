# 行为测试：爽点审计 — 蓄压爆发虚化

测试目标：验证 `shenbi-review-highpoint` 能检测出爽点虚化——前期蓄压充分但爆发被一笔带过，读者期待落空。

## 测试场景

前半章用大量篇幅蓄压（主角被围攻、身陷绝境、灵力耗尽），但高潮爆发只用了两句话就结束了，感觉"蓄了半天压，打了个寂寞"。

## 输入（章节节选）

```markdown
# 第30章 绝地反击

三名筑基后期的黑衣修士围住林轩。他背靠悬崖，灵力只剩下不到三成。

"交出剑谱，饶你不死。"领头的黑衣人冷声道。

林轩咬紧牙关。他不能交——剑谱里藏着父亲留下的秘密。

第二轮攻击到来。一道剑气擦过他的肩膀，鲜血飞溅。他勉强格挡，但步伐已经不稳。

第三轮。一道灵力击中他的胸口，他被打飞出去，重重撞在岩壁上。

他挣扎着站起来。体内灵力几乎枯竭，眼前开始发黑。

就在这时，丹田深处涌出一股陌生的力量。那是——封印在经脉中的上古传承。

林轩猛然睁眼。他一剑挥出，剑气化为数十道金色光刃，将三名黑衣人尽数斩杀。

战斗结束了。林轩单膝跪地，大口喘息。
```

## 植入的 Bug

| 位置 | 文本 | 问题 |
|------|------|------|
| P1-P7 | 蓄压阶段（7段，约250字） | 蓄压充分：围攻、绝境、受伤、灵力枯竭 |
| P8-P9 | 爆发阶段（2段，约40字） | 爆发虚化：一剑挥出就全杀了，上古传承力量的觉醒和释放仅用一句话 |

**核心问题**: 蓄压 250 字，爆发 40 字。比例 6:1，严重失衡。读者期待的是一个与蓄压等量的爆发过程（新力量的具体表现、使用代价、对手的反应），但只得到了"一剑挥出→全杀了"。

## 通过条件

- [ ] 检测出蓄压/爆发的篇幅比例严重失衡
- [ ] 标记为 `error` 级别（爽点虚化）
- [ ] 建议扩写爆发部分，添加新力量的具体表现和代价
- [ ] 评分 ≤ 4/10

## 失败条件

- 未检测出爽点虚化 → FAIL
- 认为"一剑解决很爽" → FAIL（那是叙述者的主观判断，不是读者的体验）
- 标记为 `warning` → FAIL
