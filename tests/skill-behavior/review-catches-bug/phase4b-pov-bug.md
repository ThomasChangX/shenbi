# 行为测试：POV 审计 — 信息越界

测试目标：验证 `shenbi-review-pov` 能检测出视角越界——限定 POV 角色不应知道的信息被叙述出来。

## 测试场景

本章使用林轩的第一人称限定视角。但叙述中出现苏婉在另一个房间里独自思考的内容，这是林轩不可能知道的信息。

## 准备

- `plans/chapter-20-plan.md` 明确标注: POV = 林轩，限定视角
- `novel.json` 或 `genre-config.json`: 无全知视角设置

## 输入（章节节选）

```markdown
# 第20章 各怀心事

林轩走进藏经阁，翻阅那本无名剑谱。他不知道的是，此刻在隔壁的丹房里，苏婉正看着手中的玉佩出神。

苏婉想起父亲临终前的话："这块玉佩，不到万不得已，不可示人。"她将玉佩贴在胸口，眼中闪过一丝决绝。她决定今晚就去找林轩，把一切都告诉他。

与此同时，林轩的剑谱翻到了第三页，上面赫然写着"天罡"二字。
```

## 植入的 Bug

| 位置 | 文本 | POV 限制 | 实际 |
|------|------|---------|------|
| P2-P3 | "苏婉正看着手中的玉佩出神""她决定今晚就去找林轩" | 林轩限定 POV，不可能知道隔壁房间苏婉的内心想法和决定 | 全知视角越界到苏婉的内心 |
| P2 | "他不知道的是" | 明确标注了信息边界，但随后的叙述直接越界 | "他不知道的是" ≠ 可以写出来给读者看（限定 POV 下） |

## 期望的审计输出

```markdown
## POV 审计报告

**章节**: 第20章
**结果**: 不通过

### 视角一致性
| 段落 | 问题 | POV 角色 | 越界类型 | 严重度 |
|------|------|---------|---------|--------|
| P2-P3 | 苏婉的内心独白和决定 | 林轩(限定) | 信息越界: 林轩不可能知道苏婉在看玉佩、想起父亲的话、决定找他 | error |

### 评分: 4/10 不通过
```

## 通过条件

- [ ] 检测出 P2-P3 的苏婉内心独白越界
- [ ] 标记为 `error` 级别
- [ ] 指出林轩限定 POV 不应包含其他角色的内心活动
- [ ] 评分 ≤ 5/10

## 失败条件

- 未检测出 POV 越界 → FAIL
- 认为"他不知道的是"这种句式可以解决 POV 问题 → FAIL（这是越界的标志，不是解决方案）
- 标记为 `warning` 而非 `error` → FAIL
