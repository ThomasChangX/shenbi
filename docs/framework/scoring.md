# Scoring

Three-tier scoring system:

- **T1** — skill-level: each skill output scored 0–100 against its rubric
- **T2** — phase-level: prerequisite skill scores must meet threshold
- **T3** — pipeline-level: cross-chapter continuity and integrity

Rubrics live in `tests/tiers/t1-skill/<skill>/rubric.md`. Acceptance thresholds in `tests/tiers/acceptance.json`.
