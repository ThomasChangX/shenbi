# Skills

Shenbi skills are organized into 7 pipeline phases:

1. **Genesis** — worldbuilding, power system, factions, locations
2. **Architecture** — story frame, volume map, rhythm
3. **Planning** — chapter plans, context composing
4. **Drafting** — chapter drafting, foreshadowing plant/track
5. **Audit** — style polishing, anti-detect, length normalizing
6. **Foundation** — character design, relationship map, state settling
7. **Management** — truth sync, genre config

See `skills/` for the full list.
