# Concepts

## Skills

Each skill is a `SKILL.md` file under `skills/<name>/` that instructs an AI agent on a specific novel-writing task.

## Gates

Seven validation gates (G0–G7) check pipeline state at critical boundaries.

## Scoring

T1/T2/T3 scoring uses 0–100 rubrics with per-dimension weights.

## Truth Files

Project state files (world, characters, hooks, etc.) that skills read and update.
