# 时代参考（按时代分）

此参考由 `shenbi-review-era` 技能使用。提供各时代词汇、器物、制度的核心判定方法。**不提供完整事实清单**——历史事实庞大且会更新，审计时应配合权威史料。

> **重要：** 本文件只描述**判定方法与高风险元素类型**。具体元素是否错位需要结合专业史料判断。

## 1. 中国主要朝代参考

### 1.1 词汇年代判定法

**高风险词汇类别**：
- 食物（咖啡/番茄/土豆/辣椒/蔗糖等引入朝代）
- 服饰（具体形制如"飞鱼服"等）
- 制度（官名/机构名如内阁/军机处/六部等）
- 称呼（"大人"是明清才普及）

**判定步骤**：
1. 提取章节中所有具体名词
2. 对每个名词问"这个时代有吗？"
3. 不确定 = 查阅史料
4. 史料证实存在 = pass
5. 史料证实不存在 = error
6. 史料模糊 / 学术争议 = warning（按作者设定取舍）

| 元素 | 最早出现时代 | 高风险朝代 |

## 2. 架空时代设定

### 2.1 架空类型

| 类型 | 特征 | 审计要点 |
|------|------|---------|
| 仿古架空 | 以某真实朝代为蓝本 | 仍需对齐蓝本朝代的事实 |
| 异世界架空 | 完全虚构 | 需自洽的世界规则（由 `world/rules.md` 约束） |
| 平行架空 | 真实朝代有偏差 | 需明确标记哪些是偏差 |

### 2.2 架空时代审计原则

1. 设定必须先于写作：作者在 `world/rules.md` 中声明"架空朝代" + 元素映射
2. 混用 = warning：架空 + 真实元素混用若无声明 = warning
3. 一致性优先：架空内部必须一致

## 3. 器物事实核查方法

### 3.1 高风险器物类型

- **武器**：枪械 / 火炮 / 钢铁冶炼技术
- **工具**：造纸 / 印刷 / 钟表 / 玻璃
- **服饰**：具体形制（圆领袍 / 飞鱼服 / 马褂）
- **建筑**：屋顶形式（庑殿 / 歇山 / 硬山）出现时代
- **交通**：车制 / 船型 / 桥梁技术

### 3.2 核查步骤

1. 提取章节中所有具体器物（带修饰语的名词短语）
2. 标定每个器物的"最早出现时代"
3. 与 `eraName` 对比
4. 早于时代 = error
5. 时代模糊 = warning

## 4. 地点核查方法

### 4.1 真实地点

- **核对该时代该地点的归属**：宋朝提到"燕京"应写"辽南京"或"幽州"（视情境）
- **核查地点的物理状态**：宋朝提到"长城"应意识到当时长城是另一番状态
- **核查地点的俗称演变**：同一地点不同朝代不同名

### 4.2 架空地点

- 在 `world/locations.md` 中应有明确定义
- 内部一致性（不同时期不应自相矛盾）

## 5. 制度核查方法

### 5.1 官名

- 各朝代官名差异极大
- 三省六部制（唐）/ 二府三司（宋）/ 中书省（元）/ 内阁（明）/ 军机处（清）
- 错误年代使用错误官名 = 时代错位

### 5.2 制度

- 科举制：隋唐创始，宋完善，明清僵化
- 户籍制：各朝不同
- 兵制：府兵（唐）/ 募兵（宋）/ 卫所（明）/ 八旗（清）
- 赋税制：两税法（唐）/ 一条鞭（明）/ 摊丁入亩（清）

## 6. 时代参考扩展

不同朝代 / 不同文化（欧洲 / 日本 / 其他）需单独建立参考文件。当前参考以中国朝代为主。

扩展建议：
