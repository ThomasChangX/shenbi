# 基础设定评分标准

总分 100，80+ 通过。未通过的项目需要返回修改后再审。

## 维度 1: 核心冲突 (25分)

| 分数 | 标准 |
|------|------|
| 21-25 | 三层冲突明确（表面/个人/深层），各层有独立张力 |
| 15-20 | 有两层冲突，第三层模糊 |
| 8-14 | 只有一层冲突，其余未定义 |
| 0-7 | 无明显冲突 |

## 维度 2: 开篇节奏 (20分)

| 分数 | 标准 |
|------|------|
| 17-20 | 前三章有明确 Hook + 悬念 + 第一次升级 |
| 12-16 | 有 Hook 但节奏偏慢 |
| 6-11 | 有设定但无冲突推进 |
| 0-5 | 纯设定/纯日常开头 |

## 维度 3: 世界一致性 (20分)

| 分数 | 标准 |
|------|------|
| 17-20 | 规则明确、无自相矛盾、有足够约束 |
| 12-16 | 规则存在但有模糊地带 |
| 6-11 | 规则不完整 |
| 0-5 | 无明确规则 |

## 维度 4: 角色区分度 (15分)

| 分数 | 标准 |
|------|------|
| 13-15 | 主角+2+配角有独立 voice、动机、成长弧线 |
| 9-12 | 主角突出，配角扁平但无重复 |
| 5-8 | 有配角但同质化 |
| 0-4 | 角色不可区分 |

## 维度 5: 伏笔潜力 (10分)

| 分数 | 标准 |
|------|------|
| 8-10 | 世界观包含 ≥3 个可长期培育的悬念种子 |
| 5-7 | 有 1-2 个悬念种子 |
| 2-4 | 有提及但未展开 |
| 0-1 | 无可识别的伏笔潜力 |
## 维度 6: 反套路/原创性 (10分)

| 分数 | 标准 |
|------|------|
| 8-10 | 核心设定存在 ≥1 处对题材套路的颠覆或新颖组合；套路命中数 ≤ overuse_threshold |
| 5-7 | 有一定新意但有套路依赖（命中数 > overuse_threshold，尚有 rewrite 空间）|
| 2-4 | 套路依赖明显，多数核心节拍可被 tropeInventory signatures 匹配 |
| 0-1 | 完全套路化，无可识别的原创性 |
